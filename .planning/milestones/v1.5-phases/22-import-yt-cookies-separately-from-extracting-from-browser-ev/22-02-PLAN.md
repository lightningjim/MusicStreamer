- Update status label, reset UI

`_import_from_paste(self, text)`:

- If invalid: show error "Pasted content does not appear to be valid cookies. Check the format and try again."
- If valid: write text to COOKIES_PATH, then `os.chmod(COOKIES_PATH, 0o600)`
- Update status label, reset UI

`_is_valid_cookies_txt(text)` — module-level function:
```python
def _is_valid_cookies_txt(text: str) -> bool:
    lines = [l for l in text.splitlines() if l.strip() and not l.startswith("#")]
    if not lines:
        return False
    return len(lines[0].split("\t")) == 7
```
