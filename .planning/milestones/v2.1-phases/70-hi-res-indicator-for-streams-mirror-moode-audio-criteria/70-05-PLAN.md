| Threat ID | Category | Component | Disposition | Mitigation Plan |

| T-70-14 | Tampering / Race | Stream deleted between emit and slot | mitigate | Slot fetches existing row first; returns no-op if absent (no spurious INSERT). |
| T-70-15 | Repudiation | Caps-driven DB writes not logged | accept | Phase 70 does not log per-write; the rebuilt quality_map is observable via quality_map_changed Signal subscribers + repo inspection. |
