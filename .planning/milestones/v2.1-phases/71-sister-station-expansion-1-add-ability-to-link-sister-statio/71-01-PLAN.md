CREATE TABLE IF NOT EXISTS station_siblings (
  a_id INTEGER NOT NULL,
  b_id INTEGER NOT NULL,
  FOREIGN KEY(a_id) REFERENCES stations(id) ON DELETE CASCADE,
  FOREIGN KEY(b_id) REFERENCES stations(id) ON DELETE CASCADE,

New CRUD methods (insert in Repo class, alongside other CRUD methods e.g. delete_stream around repo.py:200-230):

```python
def add_sibling_link(self, a_id: int, b_id: int) -> None:
    lo, hi = min(a_id, b_id), max(a_id, b_id)
    self.con.execute(
        "INSERT OR IGNORE INTO station_siblings(a_id, b_id) VALUES (?, ?)",
        (lo, hi),
    )
    self.con.commit()

def remove_sibling_link(self, a_id: int, b_id: int) -> None:
    lo, hi = min(a_id, b_id), max(a_id, b_id)
    self.con.execute(
        "DELETE FROM station_siblings WHERE a_id = ? AND b_id = ?",
        (lo, hi),
    )
    self.con.commit()

def list_sibling_links(self, station_id: int) -> list[int]:
    rows = self.con.execute(
        "SELECT b_id AS sid FROM station_siblings WHERE a_id = ? "
        "UNION "
        "SELECT a_id AS sid FROM station_siblings WHERE b_id = ?",
        (station_id, station_id),
    ).fetchall()
    return [r["sid"] for r in rows]
```

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-71-06 | T (Tampering) | SQL injection on a_id/b_id | mitigate | All three new methods use parameterized `con.execute(..., (lo, hi))` — no string interpolation into SQL. Standard project pattern (matches every existing Repo method). |
| T-71-07 | T (Tampering) | Bypass of CHECK(a_id < b_id) via direct caller passing equal IDs | mitigate | normalize_min_max produces lo < hi only when a_id ≠ b_id. If caller passes a_id == b_id, min/max return same value, CHECK fires, INSERT OR IGNORE swallows the IntegrityError. Defense in depth: callers (Plan 71-03 chip click, Plan 71-04 picker dialog) exclude self-id by construction. |
| T-71-08 | D (Denial of Service) | Unbounded growth of station_siblings | accept | Library size 50-200 stations; max possible link rows is N*(N-1)/2 ≈ 20k at extreme — well under any practical SQLite limit. No mitigation needed. |
| T-71-09 | I (Information Disclosure) | None — purely local SQLite, no network exposure | accept | Single-user desktop app per PROJECT.md constraints. |
