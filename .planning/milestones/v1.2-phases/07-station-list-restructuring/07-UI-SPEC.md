# Phase 7 ŌĆö UI Design Contract

> Visual and interaction contract for Phase 7: Station List Restructuring.
> Generated by gsd-ui-researcher, verified by gsd-ui-checker.

---

## Design System

| Property | Value |
|----------|-------|
| Tool | none |
| Preset | not applicable |
| Component library | GTK4 + Libadwaita (Adw 1) ŌĆö native widgets only |
| Icon library | GNOME icon theme (symbolic icons via `Gtk.Image.new_from_icon_name`) |
| Font | System default (Cantarell on GNOME) ŌĆö no override |

**Note:** This is a GTK4/Libadwaita desktop app. No web stack, no CSS framework. All design decisions follow the [Adwaita HIG](https://gnome.pages.gitlab.gnome.org/hig/). Spacing and typography values below map to GTK margin/spacing properties, not CSS tokens.

---

## Spacing Scale

GTK spacing properties in pixels (multiples of 4):

| Token | Value | GTK Property | Usage |
|-------|-------|-------------|-------|
| xs | 4px | `set_margin_*` / `spacing=4` | Row internal icon gaps, tight inline spacing |
| sm | 8px | `set_margin_*` / `spacing=8` | Filter strip margins (top/bottom/start/end), panel margins |
| md | 16px | `set_margin_*` | Section separators, dialog content padding |
| lg | 24px | `set_margin_*` | Major panel breaks |

Existing established values (from `main_window.py`, do not deviate):

- Now-playing panel: `margin_top=4, margin_bottom=4, margin_start=8, margin_end=8`
- Filter strip: `margin_top=4, margin_bottom=4, margin_start=8, margin_end=8`, `spacing=8`
- Center column: `spacing=4`

**Recently Played section header label margins:** `margin_top=8, margin_bottom=4, margin_start=12, margin_end=8`

Exceptions: `Adw.ExpanderRow` and `Adw.ActionRow` internal padding is Adwaita-controlled ŌĆö do not override with custom margins inside row widgets.

## Typography

GTK4/Libadwaita CSS classes map to these roles. Do not set explicit font sizes ŌĆö use Adwaita semantic classes only.

| Role | Adwaita Class | Usage |
|------|--------------|-------|
| Body | *(default, no class)* | Station subtitle text (provider + tags), Recently Played section header label |
| Label/Dim | `dim-label` | Idle state title label ("Nothing playing"), station name label when not playing |
| Heading | `title-3` | Active track title in now-playing panel (existing pattern) |
| Caption | `caption` | Row count or secondary metadata if needed |

Font weight: Adwaita-managed ŌĆö `title-3` is bold, all others are regular. No manual weight overrides.

Line height: Adwaita-managed. No override.

## Color

All color is Adwaita-managed via CSS classes. No hex values are hard-coded in this app.

| Role | Adwaita Mechanism | Usage |
|------|------------------|-------|
| Dominant (60%) | Window/background default | Main window background, list background |
| Secondary (30%) | `Adw.ExpanderRow` header surface | Provider group header rows |
| Accent (10%) | `suggested-action` CSS class | Stop button only (existing) |
| Destructive | `destructive-action` CSS class | Not used in this phase |

Accent reserved for: **Stop button only.** No other elements in this phase use `suggested-action` or any accent color.

The "Recently Played" section header label uses no special color class ŌĆö default foreground.

Provider group `Adw.ExpanderRow` headers use Adwaita's built-in expander row styling ŌĆö no custom color injection.

## Component Inventory

New widgets introduced in this phase:

| Widget | Type | Notes |
|--------|------|-------|
| Provider group header | `Adw.ExpanderRow` | One per provider; title = provider name; collapsed by default (`set_expanded(False)`) |
| Station row (inside group) | `Adw.ActionRow` embedded in `StationRow` (existing) | Appended as child of `Adw.ExpanderRow` via `add_row()` |
| "Uncategorized" group | `Adw.ExpanderRow` | Title = "Uncategorized"; identical visual treatment to named provider groups; positioned last |
| Recently Played section header | `Gtk.Label` | Text: "Recently Played"; no CSS class override; `margin_top=8, margin_bottom=4, margin_start=12` |
| Recently Played station rows | `StationRow` (existing) | Same widget as list rows; fully playable; appended directly to listbox above all ExpanderRows |
| Recently Played container visibility | `Gtk.Stack` or `set_visible()` toggle | Hidden when any filter is active (D-15); shown when no filter active |

**`Adw.ListBox` structure (top to bottom when no filter active):**

1. Recently Played header label (as a non-interactive `Gtk.ListBoxRow` wrapper, or inline label ŌĆö Claude's discretion on exact wrapper)
2. Up to 3 `StationRow` entries (most recent first)
3. One `Adw.ExpanderRow` per named provider (alphabetical)
4. One `Adw.ExpanderRow` for "Uncategorized" (last, only if stations exist with no provider)

## Interaction Contract

| Trigger | Response |
|---------|----------|
| Click provider group header | Toggle expand/collapse for that group; other groups unaffected |
| App launch | All provider groups collapsed; Recently Played visible (if entries exist) |
| Click station row (in group or Recently Played) | Play station; update Recently Played immediately (D-10) |
| Provider filter selected | Switch to flat ungrouped list; hide Recently Played section (D-12, D-15) |
| Search text entered (no provider filter) | Keep grouped view; hide non-matching rows within groups; hide empty groups; Recently Played hidden (D-13, D-15) |
| Both provider filter + search active | Flat list; Recently Played hidden (D-14, D-15) |
| Clear filters | Restore grouped view; show Recently Played |
| Station plays | `repo.update_last_played(station_id)` called; Recently Played section refreshed in-place (no full reload required) |

## State Transitions

| State | Recently Played | List Structure |
|-------|----------------|----------------|
| No filter, entries exist | Visible, up to 3 rows | Grouped by provider, all collapsed |
| No filter, no play history | Hidden | Grouped by provider, all collapsed |
| Provider filter active | Hidden | Flat list, filtered to selected provider |
| Search only (no provider filter) | Hidden | Grouped, non-matching rows hidden, empty groups hidden |
| Provider + search active | Hidden | Flat list, filtered |
| Empty result | Hidden | Adw.StatusPage shown (existing empty state) |
