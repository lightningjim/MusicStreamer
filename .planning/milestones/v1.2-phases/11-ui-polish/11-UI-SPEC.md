## Color

This phase uses Adwaita semantic color tokens ‚Äî no hex values hardcoded. All colors are resolved at runtime by the Adwaita theme.

| Role | Value | Usage |
|------|-------|-------|
| Dominant (60%) | `@window_bg_color` | Main window background, station list area |
| Secondary (30%) | `@card_bg_color` (Adwaita secondary surface) | Now-playing panel background |
| Accent (10%) | `@accent_bg_color` | Stop button (`.suggested-action` class only) |
| Destructive | `@destructive_bg_color` | Not used in this phase |

**Gradient specification (UI-02):**

Applied to `.now-playing-panel` CSS class via `Gtk.CssProvider`:

```css
.now-playing-panel {
    background: linear-gradient(
        to bottom,
        shade(@card_bg_color, 1.04),
        shade(@card_bg_color, 0.97)
    );
    border-radius: 12px;
}
```

- Direction: top-to-bottom
- Top stop: `shade(@card_bg_color, 1.04)` ‚Äî 4% lighter than secondary surface
- Bottom stop: `shade(@card_bg_color, 0.97)` ‚Äî 3% darker than secondary surface
- Effect: subtle depth, not decorative ‚Äî breaks flat appearance without introducing color shift
- Fallback: if `shade()` is not supported by installed GTK version, use `background-color: @card_bg_color` (flat surface still passes UI-02 intent)

**Rounded corners (UI-01):**

```css
.now-playing-panel {
    border-radius: 12px;
}
```

- Applied to now-playing panel only
- 12px radius ‚Äî matches Adwaita card radius (`.card` uses 12px)
- No border or shadow added ‚Äî surface color change provides sufficient definition

Accent reserved for: Stop button (`.suggested-action`) only. No other elements use accent in this phase.

## CSS Class Assignment

| Widget | CSS Class Added | Rule Applied |
|--------|----------------|--------------|
| `panel` (Gtk.Box in `__init__`) | `.now-playing-panel` | gradient + border-radius |
| `Adw.ActionRow` in `StationRow` and `_make_action_row` | `.station-list-row` | padding bump |
| All `Adw.ActionRow` rows (acceptable if consistent) | Alternative: target `row` globally | padding bump ‚Äî only if no regression on ExpanderRow headers |

**Preferred approach:** Add `.station-list-row` class to `Adw.ActionRow` instances in both `StationRow.__init__` and `_make_action_row`. CSS rule:

```css
.station-list-row {
    padding-top: 4px;
    padding-bottom: 4px;
}
```

**CSS loading:** Single `Gtk.CssProvider` instantiated in `__main__.py` or `MainWindow.__init__`. Loaded via `Gtk.StyleContext.add_provider_for_display(display, provider, Gtk.STYLE_PROVIDER_PRIORITY_APPLICATION)`.
