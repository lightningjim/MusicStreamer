<threat_model>
## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| Station metadata -> QLabel | Untrusted ICY/station names displayed in dialog labels |
| User input -> repo | Name/tags/URL from QLineEdit passed to repo methods |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-39-01 | Spoofing | Station name/tags in QLabel | mitigate | All QLabel instances use `Qt.PlainText` text format — prevents rich-text injection from untrusted station metadata |
| T-39-02 | Tampering | User input -> repo.update_station | accept | Input goes to SQLite parameterized queries in repo.py — no SQL injection vector; field length is bounded by QLineEdit widget |
| T-39-03 | Denial of Service | Delete while playing | mitigate | Delete button disabled when `player._current_station_name == station.name`; confirmation dialog before delete |
</threat_model>
