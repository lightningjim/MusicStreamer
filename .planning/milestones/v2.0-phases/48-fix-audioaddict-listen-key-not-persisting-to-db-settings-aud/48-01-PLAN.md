|----------|-------------|
| User input -> dialog | User clicks Clear button; no typed input flows to settings through this dialog |
| Dialog -> SQLite settings table | Repo.set_setting writes the empty string on confirmed clear |
| QMessageBox.question -> slot | Confirm gate is the security primitive for destructive action |

| Threat ID | Category | Component | Disposition | Mitigation Plan |

| T-48-02 | Repudiation | AccountsDialog._on_aa_clear_clicked | accept | Single-user desktop app; no audit log required per project convention. |
| T-48-03 | Information Disclosure | AccountsDialog._aa_status_label | mitigate | Label shows only "Saved" / "Not saved" — never renders the key material. Qt.TextFormat.PlainText prevents rich-text injection (T-40-04 parity). |
| T-48-04 | Tampering (status label) | AccountsDialog._aa_status_label | mitigate | `self._aa_status_label.setTextFormat(Qt.TextFormat.PlainText)` — matches Twitch status label contract. |
