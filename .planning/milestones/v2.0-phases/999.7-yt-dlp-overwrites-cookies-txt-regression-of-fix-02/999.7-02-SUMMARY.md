## Accomplishments

- `yt_import.scan_playlist` now wraps yt-dlp in `cookie_utils.temp_cookies_copy()` with correct nesting order (`with yt_dlp.YoutubeDL` textually inside `with cookie_utils.temp_cookies_copy`). Canonical `cookies.txt` is never passed to yt-dlp — `save_cookies()` on `__exit__` writes to the temp path, which is then unlinked in the helper's `finally`.
- Corruption check (`cookie_utils.is_cookie_file_corrupted`) runs BEFORE `opts` is built. On hit: `constants.clear_cookies()` → optional `toast_callback("YouTube cookies cleared — re-import via Accounts menu.")` → proceed without cookies. No retry loop.
- `scan_playlist` gains optional kwarg `toast_callback: Optional[Callable[[str], None]] = None` — non-breaking for existing positional callers.
- `_YtScanWorker.__init__` accepts `toast_callback` kwarg and forwards it to `scan_playlist` in `run()`. Dialog call site passes `toast_callback=self._toast` (already stored at `ImportDialog.__init__:170`).
- 5 RED tests from Plan 01 turned GREEN: `test_ytdlp_uses_temp_cookie_copy`, `test_canonical_cookies_unchanged_by_ytdlp`, `test_scan_playlist_auto_clears_corrupted`, `test_temp_cookie_unlinked_after_call`, `test_copy_failure_omits_cookiefile`.
