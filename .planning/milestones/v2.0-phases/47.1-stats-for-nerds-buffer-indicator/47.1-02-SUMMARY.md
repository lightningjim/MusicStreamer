# Phase 47.1 Plan 02: Stats-for-Nerds Buffer Indicator UI Summary

**Live GStreamer buffer-fill percent surfaced in NowPlayingPanel as a mini QProgressBar + {N}% QLabel row inside a hidden-by-default QFormLayout wrapper, toggled via a new checkable hamburger-menu QAction with SQLite persistence, and wired end-to-end to Player.buffer_percent via a bound-method connect (no lambda).**

## Performance

- **Duration:** ~9 min
- **Completed:** 2026-04-18
- **Tasks:** 4 (plus 1 Rule 3 auto-fix)
- **Files modified:** 6 (2 source + 4 test)
- **New tests:** 11 (7 panel-level + 4 integration)

## Accomplishments

- `NowPlayingPanel._build_stats_widget()` constructs a QFormLayout wrapper with "Buffer" label (muted QPalette.Disabled palette extraction per D-10) + a QProgressBar (range 0-100, `setTextVisible(False)`, `setFixedWidth(120)`) + "0%" QLabel in a QHBoxLayout (D-01/D-02).
- Wrapper added as the last center-column item (D-07) and always constructed (D-08); initial visibility reads `show_stats_for_nerds` setting with string equality (D-05, Pitfall 5 avoided).
- `set_buffer_percent(int)` slot updates both bar and label atomically (D-11).
- `set_stats_visible(bool)` slot toggles the wrapper widget (D-07).
- `MainWindow._act_stats` checkable QAction placed in its own hamburger-menu group between Accounts and Export Settings, with a new separator after (D-03, Pitfall 7). Initial checked state reads the same setting with string equality.
- `_on_stats_toggled(bool)` slot persists `"1"`/`"0"` via `repo.set_setting` AND calls `panel.set_stats_visible` (D-04, D-07).
- `self._player.buffer_percent.connect(self.now_playing.set_buffer_percent)` — bound-method wiring (D-13, QA-05, no lambda).
- 7 panel-level tests in `tests/test_now_playing_panel.py` covering D-01/D-02/D-05/D-07/D-08/D-09/D-11 (uses `isHidden()` not `isVisible()` per Pitfall 6).
- 4 integration tests in `tests/test_main_window_integration.py` covering D-03 (checkable QAction), D-04 (initial state + persistence), D-07 (panel visibility toggling), D-13 + QA-05 (functional + structural `inspect.getsource` check for no lambda on the connect line).

- **Bar fills during real playback:** During initial stream fill, the progress bar should advance 0→100% over a few seconds (queue2 low/high watermark crossings). After 100% is reached, the bar stays at 100% (D-deferred: no hide-at-full behavior).

- **Menu group isolation (D-03):** The "Stats for Nerds" item should visually appear in its own group with separators above (after Accounts) and below (before Export Settings).
