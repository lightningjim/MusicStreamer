**Bus-handler method pattern — MIRROR `_on_gst_tag`** (`player.py:291-301`):
```python
def _on_gst_tag(self, bus, msg) -> None:
    taglist = msg.parse_tag()
    found, value = taglist.get_string(Gst.TAG_TITLE)
    # Cancel the failover timer -- audio data arrived, stream is working.
    # QTimer.stop() can only be called on the owning thread, so route it
    # through a zero-delay singleShot to marshal onto the main thread.
    QTimer.singleShot(0, self._cancel_timers)
    if not found:
        return
    title = _fix_icy_encoding(value)
    self.title_changed.emit(title)  # auto-queued cross-thread to main
```

**New method shape** (insert after line 301, before `_cancel_timers` at line 307):
```python
def _on_gst_buffering(self, bus, msg) -> None:
    """Bus-loop-thread handler: parse buffer percent, emit Qt signal.
    Runs on GstBusLoopThread (not main thread). May only emit signals,
    never touch Qt widgets directly. De-dups on unchanged percent (D-14)."""
    result = msg.parse_buffering()
    # PyGObject may flatten single-out-param to bare int OR return tuple
    percent = result[0] if isinstance(result, tuple) else int(result)
    if percent == self._last_buffer_percent:
        return
    self._last_buffer_percent = percent
    self.buffer_percent.emit(percent)  # auto-queued cross-thread to main
```

**De-dup reset pattern** — insert in `_try_next_stream` after `self._current_stream = stream` (`player.py:339`):
```python
stream = self._streams_queue.pop(0)
self._current_stream = stream
# NEW: self._last_buffer_percent = -1  (reset dedup for new URL — Pitfall 3)
# Notify about failover attempt (not on first play)
if not self._is_first_attempt:
    self.failover.emit(stream)
```

**Error-recovery pattern (context only, not modified)** — `_on_gst_error` at lines 253-258 demonstrates the bus-handler → QTimer.singleShot(0, ...) pattern for work that must happen on the main thread. `_on_gst_buffering` does NOT need this — pure signal emit.

def _build_stats_widget(self) -> QWidget:
    wrapper = QWidget(self)
    form = QFormLayout(wrapper)
    form.setContentsMargins(0, 0, 0, 0)

    # Row label — muted palette (D-10, RESEARCH §6 Option A)
    buffer_row_label = QLabel("Buffer", wrapper)
    pal = buffer_row_label.palette()
    muted = pal.color(QPalette.Disabled, QPalette.WindowText)
    pal.setColor(QPalette.WindowText, muted)
    buffer_row_label.setPalette(pal)

    # Value side: progress bar + percent label inside a QHBoxLayout
    value_row = QWidget(wrapper)
    value_layout = QHBoxLayout(value_row)
    value_layout.setContentsMargins(0, 0, 0, 0)
    value_layout.setSpacing(6)
    self.buffer_bar = QProgressBar(value_row)
    self.buffer_bar.setRange(0, 100)
    self.buffer_bar.setTextVisible(False)
    self.buffer_bar.setFixedWidth(120)
    self.buffer_pct_label = QLabel("0%", value_row)
    value_layout.addWidget(self.buffer_bar)
    value_layout.addWidget(self.buffer_pct_label)
    value_layout.addStretch(1)

    form.addRow(buffer_row_label, value_row)
    return wrapper

## Shared Patterns

### Bound-method signal connections (QA-05)
**Source:** Project-wide convention, enforced across `main_window.py` and `now_playing_panel.py`.
**Apply to:** All new `.connect(...)` calls in this phase. NO self-capturing lambdas.

Example (`main_window.py:200`):
```python
self._player.title_changed.connect(self.now_playing.on_title_changed)  # GOOD
# self._player.title_changed.connect(lambda t: self.now_playing.on_title_changed(t))  # REJECTED
```

### Settings key/value idiom for string bools
**Source:** RESEARCH §5 (no existing precedent; this phase establishes the pattern).
**Apply to:** `show_stats_for_nerds` read and write sites in `now_playing_panel.py` + `main_window.py`.

```python
# Read:
show = self._repo.get_setting("show_stats_for_nerds", "0") == "1"

# Write:
self._repo.set_setting("show_stats_for_nerds", "1" if checked else "0")
```

**CRITICAL:** Never use `bool(setting_str)` — `bool("0")` is `True` (Pitfall 5). Always string-equality compare.

### Bus-handler threading contract
**Source:** `musicstreamer/player.py` (multiple sites).
**Apply to:** `_on_gst_buffering` exclusively.

Rule: bus handlers run on `GstBusLoopThread` (daemon). They MAY emit Qt Signals (auto-queued cross-thread). They MUST NOT touch `QTimer`, `QWidget`, or any Qt-affined state directly. Work that needs main-thread affinity must be marshalled via `QTimer.singleShot(0, bound_method)` — see `_on_gst_error` line 258 for the pattern. `_on_gst_buffering` needs ONLY signal emission, no marshalling.

### Test-double FakePlayer Signal surface
**Source:** `tests/test_now_playing_panel.py:28-55`, `tests/test_main_window_integration.py:31-58`.
**Apply to:** Add `buffer_percent = Signal(int)` to BOTH FakePlayer classes when tests reference it.
