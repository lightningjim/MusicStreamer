## What Was Built

**`musicstreamer/ui_qt/main_window.py`:**

**`_ImportPreviewWorker.run()` (lines ~114-124):** Added `except settings_export.TextModeCorruptionError:` clause immediately before the existing `except Exception as exc:` clause. The new clause emits `self.error.emit("TEXT_MODE_ZIP")`. The ordering is critical — TextModeCorruptionError is a ValueError subclass, so if the generic clause appeared first Python would match it and the sentinel would never be emitted. This mirrors the established `_GbsImportWorker` sentinel idiom at main_window.py:147-148 ("auth_expired").

**`_on_import_preview_error()` (lines ~1528-1546):** After the existing `self._end_busy()` first line, added a sentinel branch: `if msg == "TEXT_MODE_ZIP": QMessageBox.warning(self, "Settings import failed", "This backup looks corrupted by a text-mode file transfer. Re-copy it in binary mode and try again."); return`. The existing `truncated`/`show_toast` lines remain below the branch and handle all other errors (D-03 generic tier). `_end_busy()` runs before the branch in both paths (prevents spinner staying active during modal, Pitfall 3).

**No new imports added:** `QMessageBox` was already imported at line 40; `settings_export` already in module scope. `flatpak_import_wizard.py` untouched (D-04).
