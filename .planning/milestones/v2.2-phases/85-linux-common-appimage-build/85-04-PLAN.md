| Threat ID | Category | Component | Disposition | Mitigation Plan |

| T-85-04-02 | Information disclosure | distrobox containers see host $HOME by default | accept | Distrobox is documented behavior; smoke harness runs read-only operations against `tools/linux-build/artifacts/` and writes only to per-container `mktemp -d` tmpdirs.
