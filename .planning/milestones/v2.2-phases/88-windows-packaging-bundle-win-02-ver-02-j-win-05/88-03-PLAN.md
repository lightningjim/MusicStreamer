<threat_model>
## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| Inno installer → Win11 user shell | Installer creates/deletes the per-user Start-Menu `.lnk` and bakes the AUMID; the shell reads AUMID into the SMTC overlay and taskbar grouping. |
| GStreamer bundle → audio output | Bundled gst-libav/gst-plugins decode remote AAC streams; a missing plugin must fail loudly (exit 10), never silently produce no audio. |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-88-08 | Spoofing | SMTC overlay identity | mitigate | UAT row 3 asserts the overlay shows "MusicStreamer" not "Unknown app"; row 2 asserts the installed `.lnk` AUMID equals the canonical literal — a spoofed/blank AUMID is caught. |
| T-88-09 | Tampering | Stale taskbar-pinned `.lnk` on upgrade | mitigate | UAT row 1 confirms the Plan 01 `[InstallDelete]` removed the old `.lnk` (no stale duplicate); residual pinned-taskbar case is covered by the Plan 01 RELEASE-NOTES unpin guidance. |
| T-88-10 | Denial of Service | Missing gst-libav -> silent no-audio | mitigate | UAT row 6 confirms `check_bundle_plugins.py` exits 10 on a missing gst-libav bundle (loud fail) and AAC rows confirm audible playback on a complete bundle. |
| T-88-11 | Elevation of Privilege | Installer privilege level | accept | Installer is PrivilegesRequired=lowest, per-user; no elevation. Honest low-risk accept. |
</threat_model>
