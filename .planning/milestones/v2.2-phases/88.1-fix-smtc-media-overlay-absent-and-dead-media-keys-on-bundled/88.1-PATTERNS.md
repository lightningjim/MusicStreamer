## Shared Patterns

### WR-01: Write-Host + exit N (never Write-Error) in build.ps1
**Source:** `packaging/windows/build.ps1` lines 19-28 (docblock), lines 63-65 (example usage)
**Apply to:** New step 4c SMTC smoke guard block
```powershell
# Phase 65 WR-01: BUILD_FAIL paths use `Write-Host ... -ForegroundColor Red`
# (NOT `Write-Error`) followed by `exit N`. With $ErrorActionPreference =
# "Stop", `Write-Error` is escalated to a TERMINATING error -- the script
# unwinds through the surrounding try/finally as an unhandled exception and
# PowerShell emits its default exit code 1, never reaching the documented
# `exit N` line.
if ($LASTEXITCODE -ne 0) {
    Write-Host "BUILD_FAIL reason=... hint='...'" -ForegroundColor Red
    exit N
}
```

### Invoke-Native wrapper
**Source:** `packaging/windows/build.ps1` lines 40-50 (function definition)
**Apply to:** All native exe invocations in new build.ps1 guard blocks
```powershell
function Invoke-Native {
    param([scriptblock]$Block)
    $prev = $ErrorActionPreference
    $ErrorActionPreference = "Continue"
    try {
        & $Block | ForEach-Object {
            if ($_ -is [System.Management.Automation.ErrorRecord]) { "$($_.Exception.Message)" }
            else { $_ }
        }
    } finally { $ErrorActionPreference = $prev }
}
```

### Factory "never raises" contract
**Source:** `musicstreamer/media_keys/__init__.py` lines 23-54
