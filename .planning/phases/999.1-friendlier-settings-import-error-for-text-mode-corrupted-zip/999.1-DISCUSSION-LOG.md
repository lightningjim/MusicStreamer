## Detection precision

| Option | Description | Selected |
|--------|-------------|----------|
| Strict CRLF fingerprint | Require PK magic AND evidence of injected 0x0D0A runs before classifying as text-mode corruption. Fewest false positives. | ✓ |
| Loose (PK + BadZipFile) | PK magic + BadZipFile → assume text-mode corruption. Simpler, may mislabel genuinely truncated zips. | |

**User's choice:** Strict CRLF fingerprint
**Notes:** Prioritize avoiding false positives so a genuinely damaged/truncated binary zip isn't mislabeled as a transfer problem.
