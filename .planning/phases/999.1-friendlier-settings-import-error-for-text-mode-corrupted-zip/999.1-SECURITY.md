## Trust Boundaries

| Boundary | Description | Data Crossing |
|----------|-------------|---------------|
| file picker → `preview_import` | User-supplied `.zip` that already failed `zipfile.ZipFile` open crosses into detection | Untrusted file bytes (already known-bad zip) |

## Threat Register

| Threat ID | Category | Component | Disposition | Mitigation | Status |
|-----------|----------|-----------|-------------|------------|--------|
| T-999.1-01 | Denial of Service | `_is_text_mode_corrupted_zip` file read | mitigate | Bounded read: `_DETECT_READ_CAP = 4*1024*1024` + `fh.read(_DETECT_READ_CAP)` (`settings_export.py:69,72`); no unbounded `fh.read()`. Regression test `test_detection_bounded_read_large_file` (5 MiB → False). | closed |
| T-999.1-02 | Tampering | attacker-controlled zip internals | mitigate | Raw-byte scan only: `data[:4] == b"PK\x03\x04" and b"\x0d\x0a" in data`. No `zipfile`/decompress/member read/central-dir parse inside the helper (`settings_export.py:40-75`). | closed |
| T-999.1-03 | Info disclosure / data loss | byte rewriting | accept | No auto-repair / `0x0D0A→0x0A` collapse anywhere (D-01 forbids it). Absence-of-feature is the control; confirmed by source inspection of helper + `BadZipFile` branch (`settings_export.py:316-321`). | closed |
| T-999.1-04 | Spoofing (sentinel collision) | `error.emit("TEXT_MODE_ZIP")` | accept | Only emit site is the specific `except settings_export.TextModeCorruptionError` clause (`main_window.py:120,124`); generic `except Exception` emits `str(exc)`, which cannot equal the bare token. Mirrors established `"auth_expired"` idiom. | closed |
| T-999.1-05 | Information disclosure | dialog body text | mitigate | `QMessageBox.warning` uses fixed literal D-05 strings (`main_window.py:1539-1544`) — no f-string/`%`/`.format`/concatenation of file path or exception detail; `return` prevents fall-through to the `msg`-embedding toast. | closed |
| T-999.1-06 | Elevation / scope creep | Flatpak import wizard | mitigate | `flatpak_import_wizard.py` has zero occurrences of `TextModeCorruptionError`, `TEXT_MODE_ZIP`, or new dialog code — file untouched (D-04). | closed |

*Status: open · closed*
*Disposition: mitigate (implementation required) · accept (documented risk) · transfer (third-party)*
