Create `musicstreamer/settings_export.py` with these components:

**Dataclasses** (at module top):
```python
@dataclass
class ImportDetailRow:
    name: str
    action: str  # "add" | "replace" | "skip" | "error"
    reason: str = ""

@dataclass
class ImportPreview:
    added: int = 0
    replaced: int = 0
    skipped: int = 0
    errors: int = 0
    detail_rows: list = field(default_factory=list)     # List[ImportDetailRow]
    track_favorites: list = field(default_factory=list)  # raw dicts from JSON
    settings: list = field(default_factory=list)          # raw dicts from JSON
    zip_path: str = ""
    stations_data: list = field(default_factory=list)     # raw station dicts for commit
```

**`_sanitize(name: str) -> str`**: Unicode NFKD normalize, ASCII-encode (ignore errors), strip non-word/space/dot/dash, collapse whitespace to underscore, truncate to 80 chars, fallback to "station" if empty.

**`_station_to_dict(station: Station) -> dict`**: Serialize Station + its streams list to the JSON schema from RESEARCH.md. Fields: name, provider (provider_name), tags, icy_disabled, is_favorite, last_played_at, logo_file (set during ZIP write), streams (list of dicts with url, label, quality, position, stream_type, codec).

**`_fav_to_dict(fav: Favorite) -> dict`**: Serialize Favorite to dict with station_name, provider_name, track_title, genre, created_at.

**`build_zip(repo: Repo, dest_path: str) -> None`**:
1. `repo.list_stations()` to get all stations (includes streams).
2. `repo.list_favorites()` for track favorites.
3. Query settings: `repo.con.execute("SELECT key, value FROM settings").fetchall()` — exclude keys in `_EXCLUDED_SETTINGS = {"audioaddict_listen_key"}` (per D-05/SYNC-02).
4. Build station dicts via `_station_to_dict`. For each station with `station_art_path`, resolve absolute path via `os.path.join(paths.data_dir(), station_art_path)` (if not already absolute). If file exists, set `logo_file = f"logos/{_sanitize(station.name)}{ext}"` on the dict. If collision (same sanitized name), append `_{station.id}` to disambiguate.
5. Assemble payload dict: `{"version": 1, "exported_at": datetime.datetime.now().isoformat(), "stations": [...], "track_favorites": [...], "settings": [...]}`.
6. Write ZIP with `zipfile.ZipFile(dest_path, "w", zipfile.ZIP_DEFLATED)`: writestr "settings.json", then write each logo file.

**`preview_import(zip_path: str, repo: Repo) -> ImportPreview`**:
1. Open ZIP — catch `zipfile.BadZipFile` and raise `ValueError("Not a valid ZIP archive")`.
2. Security: iterate `zf.infolist()`, reject any member where `filename.startswith('/') or '..' in filename` with `ValueError(f"Unsafe path in archive: {filename}")`.
3. Check `"settings.json" in zf.namelist()` — raise `ValueError("Missing settings.json")` if absent.
4. Parse JSON, check `payload.get("version") != 1` — raise `ValueError(f"Unsupported version: {payload.get('version')}")`.
5. For each station in `payload["stations"]`: check if ANY stream URL exists via `repo.station_exists_by_url(stream["url"])` for each stream. If match found: action="replace", get existing station_id. If no match: action="add". Build `ImportDetailRow` for each.
6. Return `ImportPreview` with counts, detail_rows, track_favorites, settings, zip_path, stations_data.

**`commit_import(preview: ImportPreview, repo: Repo, mode: str) -> None`** (`mode` is `"merge"` or `"replace_all"`):
1. Wrap everything in `with repo.con:` for atomicity.
2. If `mode == "replace_all"`: execute `DELETE FROM station_streams`, `DELETE FROM stations`, `DELETE FROM favorites`, `DELETE FROM providers` (in that order — FK ordering per Pitfall 3).
3. Open `zipfile.ZipFile(preview.zip_path, "r")` for logo extraction.
4. For each station in `preview.stations_data` paired with its `detail_row`:
   - If `action == "add"` or (`action == "replace"` and mode is replace_all which already cleared):
     - `provider_id = repo.ensure_provider(station["provider"])` if provider present.
     - Insert station: `repo.con.execute("INSERT INTO stations(name, provider_id, tags, icy_disabled, last_played_at, is_favorite) VALUES(?,?,?,?,?,?)", ...)` — get `station_id = cur.lastrowid`.
     - Insert each stream: `repo.con.execute("INSERT INTO station_streams(station_id, url, label, quality, position, stream_type, codec) VALUES(?,?,?,?,?,?,?)", ...)`.
     - Set favorite flag: `repo.con.execute("UPDATE stations SET is_favorite=? WHERE id=?", (int(station.get("is_favorite", False)), station_id))`.
   - If `action == "replace"` (merge mode — URL match found):
     - Find existing station_id via `repo.con.execute("SELECT ss.station_id FROM station_streams ss WHERE ss.url=?", (matched_url,)).fetchone()`.
     - `repo.con.execute("DELETE FROM station_streams WHERE station_id=?", (station_id,))` — clear stale streams (Pitfall 2).
     - Update station: `repo.con.execute("UPDATE stations SET name=?, provider_id=?, tags=?, icy_disabled=?, last_played_at=?, is_favorite=? WHERE id=?", ...)`.
     - Re-insert all streams from import data.
   - Extract logo: if `station.get("logo_file")` and that entry exists in ZIP, extract to `{paths.data_dir()}/assets/{station_id}/station_art{ext}`. Create directory with `os.makedirs(..., exist_ok=True)`. Update art path: `repo.con.execute("UPDATE stations SET station_art_path=? WHERE id=?", (f"assets/{station_id}/station_art{ext}", station_id))`.
5. Favorites: for each fav in `preview.track_favorites`: `repo.con.execute("INSERT OR IGNORE INTO favorites(station_name, provider_name, track_title, genre, created_at) VALUES(?,?,?,?,?)", ...)`.
6. Settings: for each setting in `preview.settings`: `repo.con.execute("INSERT OR REPLACE INTO settings(key, value) VALUES(?,?)", (s["key"], s["value"]))`.

**IMPORTANT**: Use `repo.con.execute()` directly inside the `with repo.con:` block for atomicity — do NOT call `repo.insert_station()` / `repo.add_favorite()` etc. because those methods call `repo.con.commit()` internally which would break the single-transaction guarantee.

Write tests in `tests/test_settings_export.py` following the behavior list above. Use `tmp_path` fixture for ZIP file paths and in-memory DB.
