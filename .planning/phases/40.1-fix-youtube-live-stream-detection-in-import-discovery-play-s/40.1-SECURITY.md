## Accepted Risks Log

The following `accept` threats have their rationale verified in code; no new runtime mitigation required:

- **T-40.1-YT-01**: YT scan payloads rendered as plain text via QListWidget default.
- **T-40.1-DIS-01**: Icon paths are compile-time constants in `discovery_dialog.py`.
- **T-40.1-FB-01**: Import counts coerced via `int()` and rendered via `QLabel.setText`.
- **T-40.1-LOGO-02**: Outbound image fetches target user-imported YT/AA domains only.
- **T-40.1-ICY-01**: `icy_label.setTextFormat(Qt.PlainText)` present.
- **T-40.1-TIMER-02**: Python int counter is unbounded.
