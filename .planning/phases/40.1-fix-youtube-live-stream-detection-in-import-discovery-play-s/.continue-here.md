## Critical Anti-Patterns

| Pattern | Description | Severity | Prevention Mechanism |
|---------|-------------|----------|---------------------|
| Parallel worktree dispatch in a single message | Sending multiple `Task(run_in_background:true, isolation:"worktree")` calls in the same assistant turn races on `.git/config.lock` | advisory | Dispatch one worktree Task per turn; natural round-trip latency serializes `git worktree add` |
| Post-merge `git commit --amend` of STATE.md/ROADMAP.md | Wave 1 merge script rewound main HEAD (commits recovered via reflog, but disruptive). The amend block combined with `git worktree remove` caused cwd loss. | advisory | Skip the amend step — do simple `git merge --no-edit`, then `git worktree remove --force`, then `git branch -D`. Update STATE/ROADMAP separately via `gsd-tools roadmap update-plan-progress`. |
