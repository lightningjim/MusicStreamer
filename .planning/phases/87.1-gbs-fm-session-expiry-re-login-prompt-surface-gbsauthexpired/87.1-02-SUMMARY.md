### Auto-resolved Architectural Note

- **Issue:** `_fetch_marquee` is a module-level function with no `self` reference. The plan's suggested emission site (`self.auth_expired.emit()` inside `_fetch_marquee`) was not feasible as written.
- **Plan guidance:** The plan explicitly anticipated this: "If `_fetch_marquee` is a module-level function (not a method) and lacks a `self`, instead emit from the worker's run-loop wrapper that calls `_fetch_marquee` â€” check the call site."
