Create musicstreamer/ui/import_dialog.py with class ImportDialog(Adw.Window).

**Constructor (__init__):**
- Args: `app`, `repo: Repo`, `main_window`
- `self.set_default_size(700, 560)`, `set_transient_for(main_window)`, `set_modal(True)`
- Title: "Import YouTube Playlist"
- Instance vars: `self.repo = repo`, `self.main_window = main_window`, `self._checklist_items: list[tuple[Gtk.CheckButton, dict]] = []`

**Layout (Adw.ToolbarView):**
- Top bar: `Adw.HeaderBar()`
- Content: vertical Gtk.Box with margin 12 all sides, spacing 8

**Top row (horizontal Gtk.Box, spacing 8):**
- `self._url_entry = Gtk.Entry()` with placeholder "Paste YouTube playlist URL...", hexpand=True
- `self._url_entry.connect("changed", self._on_url_changed)` — enables/disables scan button
- `self._scan_btn = Gtk.Button(label="Scan Playlist")`
- `self._scan_btn.set_sensitive(False)` (disabled until URL entered)
- `self._scan_btn.connect("clicked", self._on_scan_clicked)`

**Gtk.Stack with 4 named pages:**
1. `"prompt"` — `Adw.StatusPage(title="Paste a playlist URL", description="Enter a public YouTube playlist URL above and tap Scan Playlist.")`
2. `"scanning"` — centered Gtk.Box with `Gtk.Spinner(spinning=True)` size 32x32
3. `"error"` — `Adw.StatusPage()` (title/description set dynamically)
4. `"checklist"` — vertical Gtk.Box containing:
   - `Gtk.ScrolledWindow(vexpand=True)` > `self._listbox = Gtk.ListBox(selection_mode=NONE)`
   - `self._progress_label = Gtk.Label(label="")` with `set_visible(False)`
   - Footer horizontal Gtk.Box (spacing 8):
     - `self._count_label = Gtk.Label()` with hexpand=True, xalign=0
     - `self._import_btn = Gtk.Button(label="Import Selected")` with CSS class `suggested-action`
     - `self._import_btn.connect("clicked", self._on_import_clicked)`

**_on_url_changed(self, entry):**
- `self._scan_btn.set_sensitive(bool(entry.get_text().strip()))`

**_on_scan_clicked(self, btn):**
- Get URL from entry, validate with `is_yt_playlist_url(url)`. If invalid, show error page: title="Invalid URL", description="That doesn't look like a YouTube playlist URL. Check the link and try again."
- Set stack to "scanning", disable entry + scan btn
- Start daemon thread calling `self._scan_worker(url)`

**_scan_worker(self, url):**
- try: `entries = scan_playlist(url)`, then `GLib.idle_add(self._on_scan_complete, entries)`
- except ValueError: `GLib.idle_add(self._on_scan_error, "Playlist Not Accessible", "This playlist may be private or unavailable. Try a public playlist URL.")`
- except (RuntimeError, subprocess.TimeoutExpired): `GLib.idle_add(self._on_scan_error, "Could Not Reach YouTube", "Check your internet connection and try again.")`
- except Exception as e: `GLib.idle_add(self._on_scan_error, "Scan Failed", str(e))`

**_on_scan_complete(self, entries):**
- If empty: show error page with title="No Live Streams Found", description="This playlist has no live streams. Only live broadcasts can be imported as stations." Re-enable entry + scan btn. Return.
- Clear listbox, clear self._checklist_items
- For each entry: create Adw.ActionRow with title=GLib.markup_escape_text(entry["title"]), subtitle=GLib.markup_escape_text(entry["provider"]), Gtk.CheckButton prefix (active=True). Append (check, entry) to self._checklist_items. Append row to listbox.
- Set count label: f"{len(entries)} live stream{'s' if len(entries) != 1 else ''} found"
- Set stack to "checklist"
- Re-enable entry + scan btn

**_on_scan_error(self, title, description):**
- Set error StatusPage title and description, show "error" page
- Re-enable entry + scan btn

**_on_import_clicked(self, btn):**
- Collect selected: `selected = [entry for (check, entry) in self._checklist_items if check.get_active()]`
- If not selected: return
- Disable import_btn, scan_btn, url_entry
- Show progress_label, set text "0 imported, 0 skipped"
- Start daemon thread: `self._import_worker(selected)`

**_import_worker(self, selected):**
- `def on_progress(imp, skip): GLib.idle_add(self._update_progress, imp, skip)`
- `imported, skipped = import_stations(selected, self.repo, on_progress=on_progress)`
- `GLib.idle_add(self._on_import_done, imported, skipped)`

**_update_progress(self, imported, skipped):**
- `self._progress_label.set_text(f"{imported} imported, {skipped} skipped")`

**_on_import_done(self, imported, skipped):**
- Update progress label with final count
- Change import_btn label to "Done", re-enable it
- Disconnect old import_btn handler, connect new: `self._import_btn.connect("clicked", self._on_done_clicked)`

**_on_done_clicked(self, btn):**
- `self.main_window.reload_list()`
- `self.close()`
