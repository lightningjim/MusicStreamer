**Add tag chip toggled callback:**
```python
def _on_tag_chip_toggled(self, btn, tag_name):
    if self._rebuilding:
        return
    if btn.get_active():
        self._selected_tags.add(tag_name)
    else:
        self._selected_tags.discard(tag_name)
```
