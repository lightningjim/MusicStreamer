### Auto-fixed Issues

**1. [Rule 1 - Bug] Station is unhashable — set comprehension in test_discover_row_map_combo_includes_unflagged_station**
- **Found during:** Task 3 (GREEN run)
- **Issue:** The test used `{map_combo.itemData(i, Qt.UserRole) for i in range(...)}` — a set comprehension — but `Station` is a dataclass without `__hash__`, so `TypeError: cannot use 'musicstreamer.models.Station' as a set element` was raised.
- **Fix:** Changed to a list comprehension `[map_combo.itemData(i, Qt.UserRole) for i in range(...)]` and kept `assert stationFlagged in combo_data` (list membership check works without hashing).
- **Files modified:** `tests/test_live_refresh_dialog.py`
