class _SomaImportWorker(QThread):
    import_finished = Signal(int, int)
    error = Signal(str)
    def run(self):
        try:
            from musicstreamer.repo import Repo
            from musicstreamer import soma_import
            channels = soma_import.fetch_channels()
            repo = Repo(db_connect())
            inserted, skipped = soma_import.import_stations(channels, repo)
            self.import_finished.emit(int(inserted), int(skipped))
        except Exception as exc:
            self.error.emit(str(exc))

<threat_model>
## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| SomaFM API (`channels.json`) → DB | upstream preroll URLs fetched + inserted by the re-fetch worker |
| QThread worker → SQLite | re-fetch writes off the main thread |
| user → OS file viewer | "Open preroll log" hands a local file path to the OS default app |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-90-06 | Tampering | upstream preroll URL inserted by manual re-fetch | mitigate | Worker inserts ONLY via `repo.insert_preroll` (T-83-01 scheme gate rejects non-HTTP(S)); `ValueError` caught + skipped. SomaFM API endpoint is fixed (`api.somafm.com/channels.json`), no user-controlled URL. |
| T-90-07 | Tampering (concurrency) | SQLite write from QThread | mitigate | Pattern 4 thread-local Repo via `db_connect()` inside `run()`, closed in `finally` (Pitfall 7); never shares the main-thread Repo. |
| T-90-08 | Denial of Service | duplicate concurrent re-fetch workers | mitigate | SYNC-05 double-click guard (`_soma_refetch_worker is not None`) + worker retention prevents concurrent runs / GC of a running QThread. |
| T-90-09 | Information Disclosure | "Open preroll log" hands a path to the OS | accept | Local diagnostic file under the user's own data dir; `QUrl.fromLocalFile` + `os.path.isfile` guard; no path injection (path is a fixed helper, not user input). |
| T-90-SC | Tampering | npm/pip/cargo installs | accept | No package installs — existing project modules + PySide6 only (RESEARCH Package Legitimacy Audit: none). |
</threat_model>
