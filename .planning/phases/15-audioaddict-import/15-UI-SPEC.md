# Phase 15 — UI Design Contract

> Visual and interaction contract for the AudioAddict Import phase. Generated by gsd-ui-researcher.

---

## Design System

| Property | Value |
|----------|-------|
| Tool | none — GTK4 / libadwaita (Adw) |
| Preset | not applicable |
| Component library | libadwaita 1.x (Adw.Window, Adw.HeaderBar, Adw.ToolbarView, Adw.StatusPage, Adw.ActionRow, Adw.ToggleGroup) |
| Icon library | GNOME symbolic icons via Gtk.Image (icon-name) |
| Font | System default (Cantarell on GNOME) — inherited from Adwaita theme |

No web design system. All styling via GTK CSS loaded in `__main__.py` and widget properties. Do not introduce external stylesheets.

## Spacing Scale

All values align to the 4-point grid already in use in the codebase.

| Token | Value | Usage |
|-------|-------|-------|
| xs | 4px | Row padding top/bottom (`.station-list-row`, `.favorites-list-row`) |
| sm | 8px | Default `spacing=` in Gtk.Box, margin-start/end on filter bars |
| md | 12px | Dialog content margin (all four sides — matches `import_dialog.py`) |
| lg | 16px | Panel margin-top/bottom (now-playing panel) |
| xl | 24px | Panel margin-start/end (now-playing panel) |

Exceptions:
- Dialog default size: 700 × 560 px — matches existing `ImportDialog.set_default_size(700, 560)`
- Spinner size: 32 × 32 px — matches existing scanning spinner in `import_dialog.py`
- Tab strip: use `Gtk.Notebook` or `Adw.ViewSwitcherBar` native sizing — no manual override

## Typography

This app inherits all type from the Adwaita/system theme. Declare roles for consistency; do not hardcode font sizes via CSS.

| Role | GTK mechanism | Weight | Line Height |
|------|--------------|--------|-------------|
| Body | `Gtk.Label` default | Regular (400) | 1.5 (theme default) |
| Label / sublabel | `Adw.ActionRow` subtitle | Regular (400) | 1.4 (theme default) |
| Heading (dialog title) | `Adw.HeaderBar` title | Semibold (600) | 1.2 (theme default) |
| Status heading | `Adw.StatusPage` title | Semibold (600) | 1.2 (theme default) |

Do not set `set_markup()` or custom font sizes except where required for label hierarchy (e.g., `Adw.StatusPage` description uses body weight).

## Color

All colors are resolved at runtime from the Adwaita named palette. No hardcoded hex values.

| Role | Adwaita token | Usage |
|------|--------------|-------|
| Dominant (60%) | `@window_bg_color` | Dialog background, content area |
| Secondary (30%) | `@card_bg_color` | Checklist rows (Gtk.ListBox), tab content area |
| Accent (10%) | `@accent_color` | Primary action button (`suggested-action` CSS class) only |
| Destructive | `@destructive_color` | Not used in this phase — no destructive actions |

Accent reserved for: the single primary CTA button only ("Import" button with `add_css_class("suggested-action")`). All other interactive elements use default Adwaita styling.

Now-playing panel gradient uses the existing `.now-playing-panel` CSS rule (`shade(@card_bg_color)`) — no change to this rule in Phase 15.

## Interaction States

### AudioAddict Tab — State Machine

| State | Trigger | UI |
|-------|---------|-----|
| `prompt` | Tab open, no key saved | StatusPage: "Enter your AudioAddict API key", Import button disabled |
| `prompt-filled` | Key present (pre-filled or typed) | Import button enabled, quality toggle active |
| `importing` | Import button clicked | Spinner visible, progress label updating, Import button + entry disabled |
| `done` | Import worker completes | Progress label shows final count, Import button becomes "Done" |
| `error-invalid-key` | API returns 401/403 | Inline error label below entry: "Invalid or expired API key. Check your key and try again." |
| `error-network` | Network failure | StatusPage: title + description (see Copywriting) |

### Tab Persistence
- Dialog remembers last-active tab using a module-level or instance variable; defaults to "YouTube Playlist" on first open
- API key pre-filled from SQLite settings on open if previously saved
- Quality pre-filled from SQLite settings (default: "Hi" if never saved)

## Layout — Unified ImportDialog (Refactored)

```
Adw.Window (700 × 560)
  Adw.ToolbarView
    Adw.HeaderBar  [title: "Import"]
    Gtk.Box (vertical, spacing=8, margin=12)
      Gtk.Notebook
        Tab 0: "YouTube Playlist"  [existing YouTube tab content, unchanged]
        Tab 1: "AudioAddict"
          Gtk.Box (vertical, spacing=8)
            Gtk.Box (horizontal, spacing=8)   [key row]
              Gtk.Entry  [API key entry, hexpand=True]
            Adw.ToggleGroup  [Hi | Med | Low]
            Gtk.Label  [inline error, hidden by default]
            Gtk.Stack (vexpand=True)
              page "prompt"  → Adw.StatusPage
              page "importing" → Gtk.Box (spinner centered)
              page "error" → Adw.StatusPage
            Gtk.Box (horizontal, spacing=8)  [footer]
              Gtk.Label  [progress label, hexpand=True, xalign=0]
              Gtk.Button [Import / Done, suggested-action]
```
