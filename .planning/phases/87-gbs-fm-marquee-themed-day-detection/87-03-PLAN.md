<objective>
Plan 87-03 implements `GbsMarqueeWorker` — the long-lived QThread that owns its own QTimer and drives the 60s/5min/idle cadence state machine. This is the architectural keystone of Phase 87: it wires Phase 76 cookies-jar auth to Phase 87 marquee polling, emits typed PyQt signals for results, and routes D-18 quiet failures to a buffer_log-style structured WARN sink without surfacing to the UI.

The worker DIVERGES from `_AaLiveWorker`'s per-cycle spawn pattern (the closest precedent): `_AaLiveWorker` runs once per poll and dies; `GbsMarqueeWorker` is constructed once at app start, stays alive for the app lifetime, and changes cadence via a thread-safe Signal bridge (Pitfall #7). The bridge pattern is the critical correctness lever — without `self.exec_()` in `run()`, the worker-thread QTimer never fires.

Pitfall #7 reading is MANDATORY before implementation. The sketch in 87-RESEARCH.md lines 596-651 is the canonical shape.

```python
class GbsMarqueeWorker(QThread):
    themed_logo_ready = Signal(object)   # QPixmap or None — Plan 87-04 emits this
    marquee_ready = Signal(str, str)     # (first_segment, full_text)
    cadence_changed_internal = Signal(int)  # ms — cross-thread bridge

    def __init__(self, parent=None):
        super().__init__(parent)
        self._timer: QTimer | None = None
        self._themed_day_detected_this_session: bool = False
        self._last_full_marquee_text: str = ""
        self._interval_ms: int = 0
        self.cadence_changed_internal.connect(
            self._apply_cadence_on_worker_thread, Qt.QueuedConnection
        )

    def set_cadence(self, ms: int) -> None:
        """Public: callable from main thread. 0 = pause; >0 = restart interval."""
        self.cadence_changed_internal.emit(int(ms))

    def _apply_cadence_on_worker_thread(self, ms: int) -> None:
        if self._timer is None:
            self._timer = QTimer()
            self._timer.setSingleShot(True)
            self._timer.timeout.connect(self._on_tick)
        if ms == 0:
            self._timer.stop()
        else:
            self._interval_ms = ms
            self._timer.start(0)  # immediate first tick after cadence change

    def _on_tick(self) -> None:
        # ... fetch + parse + emit marquee_ready / themed_logo_ready ...
        if self._timer is not None and self._interval_ms > 0:
            self._timer.start(self._interval_ms)

    def run(self):
        self.exec_()  # CRITICAL: without this, the worker-thread QTimer never fires
```
