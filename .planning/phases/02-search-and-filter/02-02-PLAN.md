    **New methods to add:**

    `_filter_func(self, row)`:
    - Get `station = row.station`
    - Read `search_text = self.search_entry.get_text()`
    - Read provider: `prov_idx = self.provider_dropdown.get_selected()` -- if > 0, `provider_filter = self._provider_items[prov_idx]`, else `provider_filter = None`
    - Read tag: `tag_idx = self.tag_dropdown.get_selected()` -- if > 0, `tag_filter = self._tag_items[tag_idx]`, else `tag_filter = None`
    - Call `result = matches_filter(station, search_text, provider_filter, tag_filter)`
    - If result is True, increment `self._visible_count`
    - Return result

    `_on_filter_changed(self, *_)`:
    - If `self._rebuilding`: return immediately
    - Set `self._visible_count = 0`
    - Call `self.listbox.invalidate_filter()`
    - Call `self._update_clear_button()`
    - Call `self._update_empty_state()`

    `_any_filter_active(self) -> bool`:
    - Return True if `self.search_entry.get_text()` is non-empty OR `self.provider_dropdown.get_selected() > 0` OR `self.tag_dropdown.get_selected() > 0`

    `_update_clear_button(self)`:
    - `self.clear_btn.set_visible(self._any_filter_active())`

    `_update_empty_state(self)`:
    - If `self._visible_count == 0` and `self._any_filter_active()`: `self.shell.set_content(self.empty_page)`
    - Else: `self.shell.set_content(self.scroller)`

    `_on_clear(self, *_)`:
    - `self.search_entry.set_text("")`
    - `self.provider_dropdown.set_selected(0)`
    - `self.tag_dropdown.set_selected(0)`
    - Call `self._on_filter_changed()` (in case signals don't fire for programmatic resets)

    `_rebuild_filter_state(self)`:
    - Set `self._rebuilding = True`
    - Get all stations: `stations = self.repo.list_stations()`
    - Build provider list: `providers = sorted({s.provider_name for s in stations if s.provider_name})`
    - `self._provider_items = ["All Providers"] + providers`
    - `self.provider_dropdown.set_model(Gtk.StringList.new(self._provider_items))`
    - Build tag vocabulary: collect all normalized tags across all stations, deduplicate case-insensitively, sort alphabetically
      ```python
      all_tags = {}
      for s in stations:
          for t in normalize_tags(s.tags):
              key = t.casefold()
              if key not in all_tags:
                  all_tags[key] = t
      self._tag_items = ["All Tags"] + sorted(all_tags.values(), key=str.casefold)
      self.tag_dropdown.set_model(Gtk.StringList.new(self._tag_items))
      ```
    - Set `self._rebuilding = False`

    **Modify existing `reload_list` method:**
    - After the existing loop that appends StationRow children, add a call to `self._rebuild_filter_state()`
    - This ensures dropdowns stay in sync after station edits

    **Initialization order in __init__:**
    Ensure `self._rebuilding = False` and `self._visible_count = 0` are set before `self.reload_list()` is called. The `reload_list()` call at the end of __init__ will trigger `_rebuild_filter_state()` which populates the dropdowns.
