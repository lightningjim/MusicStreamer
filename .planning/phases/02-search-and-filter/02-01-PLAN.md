<feature>
  <name>Filter utilities: normalize_tags and matches_filter</name>
  <files>musicstreamer/filter_utils.py, tests/test_filter_utils.py</files>
  <behavior>
    normalize_tags:
    - "Lofi, Chill" -> ["Lofi", "Chill"]
    - "lofi • chill • Jazz" -> ["lofi", "chill", "Jazz"]
    - "Lofi, lofi, LOFI" -> ["Lofi"] (first-seen display form kept)
    - "" -> []
    - "  , , " -> []
    - "Rock • ,  Blues , Rock" -> ["Rock", "Blues"] (mixed delimiters, dedup)

    matches_filter (Station, search_text, provider_filter, tag_filter):
    - ("", None, None) -> True for any station (no filters active)
    - ("soma", None, None) -> True if "soma" is substring of station.name (case-insensitive)
    - ("soma", None, None) -> False if station.name is "DI.fm Lounge"
    - ("", "Soma.FM", None) -> True if station.provider_name == "Soma.FM"
    - ("", "Soma.FM", None) -> False if station.provider_name == "DI.fm"
    - ("", None, "lofi") -> True if "lofi" in normalized tags (case-insensitive)
    - ("", None, "lofi") -> False if station has tags "Rock, Blues"
    - ("groove", "Soma.FM", "lofi") -> True only if ALL three match
    - ("groove", "Soma.FM", "lofi") -> False if name matches but provider doesn't
    - Provider filter None or "" means inactive
    - Tag filter None or "" means inactive
    - Search text "" means inactive
  </behavior>
  <implementation>
    Create musicstreamer/filter_utils.py with two functions:

    1. normalize_tags(raw: str) -> list[str]:
       - Split raw string on regex pattern [,\u2022] (comma and bullet)
       - Strip whitespace from each token
       - Discard empty tokens
       - Deduplicate case-insensitively using casefold() as key, preserving first-seen display form
       - Return list of unique display-form strings

    2. matches_filter(station: Station, search_text: str, provider_filter: str | None, tag_filter: str | None) -> bool:
       - If search_text is non-empty: return False if search_text.casefold() not in station.name.casefold()
       - If provider_filter is non-empty and not None: return False if station.provider_name != provider_filter
       - If tag_filter is non-empty and not None: return False if tag_filter.casefold() not in {t.casefold() for t in normalize_tags(station.tags)}
       - Return True (all active filters matched)

    Import: `import re` for split, `from musicstreamer.models import Station` for type hint only.
    Do NOT import any GTK modules.
  </implementation>
</feature>
