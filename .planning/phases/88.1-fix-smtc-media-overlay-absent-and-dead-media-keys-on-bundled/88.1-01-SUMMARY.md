## Decisions Made

- `type(e).__name__` in WARNING (not `exc_info=True`) per RESEARCH Pitfall 8 — stack trace is noisy on dev Windows environments without pywinrt; the class name alone is sufficient for diagnosis
- `QCoreApplication` (not `QApplication`) in `_run_check_mediakeys` — `QApplication` requires a display and crashes on headless build machines
- Exit code 1 on Linux is CORRECT and expected — NoOpMediaKeysBackend is returned on Linux; the success path (exit 0) is VM-only and deferred to 88-03
- TDD RED tests written as a block before Task 1 implementation; they cover both Task 1's factory behavior and Task 3's test additions simultaneously
