    Create `musicstreamer/ui/discovery_dialog.py` with class `DiscoveryDialog(Adw.Window)`.

    **Constructor `__init__(self, app, repo: Repo, main_window)`:**
    - `super().__init__(application=app, title="Discover Stations")`
    - `self.repo = repo`, `self.main_window = main_window`
    - `self.set_default_size(700, 560)` (per UI-SPEC)
    - `self.set_transient_for(main_window)` and `self.set_modal(True)`
    - State: `self._debounce_id = None`, `self._cancelled = False`, `self._preview_station = None`
    - Save prior station: `self._prior_station = main_window._current_station`
    - `self._saved_urls = set()` — track URLs saved this session to disable save buttons

    **Widget hierarchy** (per UI-SPEC):
    ```
    Adw.ToolbarView
      Adw.HeaderBar (title: "Discover Stations")
      Gtk.Box (vertical, spacing=8, margins: top=12, bottom=12, start=12, end=12)
        Gtk.Box (horizontal, spacing=8) — search row
          Gtk.SearchEntry (hexpand=True, placeholder="Search by name...")
          Gtk.DropDown (tag filter, width_request=140)
          Gtk.DropDown (country filter, width_request=140)
        Gtk.Stack
          "prompt"   -> Adw.StatusPage (title="Search for stations", description="Type a name above to search Radio-Browser.info")
          "loading"  -> Gtk.Box with centered Gtk.Spinner (pixel_size=32, spinning=True)
          "results"  -> Gtk.ScrolledWindow -> Gtk.ListBox
          "empty"    -> Adw.StatusPage (title="No stations found", description="Try different search terms or remove filters.")
          "error"    -> Adw.StatusPage (title="Could not reach Radio-Browser", description="Check your internet connection and try again.") with "Retry Search" button child
    ```

    **Search entry:**
    - Connect `search-changed` signal to `_on_search_changed`
    - `_on_search_changed`: if `self._debounce_id is not None`, call `GLib.source_remove(self._debounce_id)`. Then `self._debounce_id = GLib.timeout_add(500, self._fire_search)`.
    - `_fire_search`: set `self._debounce_id = None`, call `self._do_search()`, return `False`

    **Filter dropdowns:**
    - Tag dropdown: `Gtk.StringList` with index 0 = "Any genre". Populate in background thread on dialog open via `radio_browser.fetch_tags(200)`. On result, append tag names to StringList via `GLib.idle_add`.
    - Country dropdown: `Gtk.StringList` with index 0 = "Any country". Populate in background thread on dialog open via `radio_browser.fetch_countries()`. Store `self._country_codes = [""]` list parallel to StringList. On result, append display names and store iso codes.
    - Connect `notify::selected` on both dropdowns to `_on_filter_changed`
    - `_on_filter_changed`: call `self._do_search()` immediately (no debounce — per UI-SPEC, explicit user choice)

    **`_do_search(self)`:**
    - Get search text: `self._search_entry.get_text().strip()`
    - If search text is empty and both filters are at index 0, switch stack to "prompt" and return (Pitfall 5: don't search empty)
    - Switch stack to "loading"
    - Get selected tag: if tag dropdown index > 0, get string at that index, else `""`
    - Get selected countrycode: if country dropdown index > 0, get `self._country_codes[index]`, else `""`
    - Spawn daemon thread calling `radio_browser.search_stations(name, tag, countrycode)`
    - On success: `GLib.idle_add(self._on_results, results)`
    - On exception: `GLib.idle_add(self._on_error, str(e))`

    **`_on_results(self, results)`:**
    - If `self._cancelled`: return (Pitfall 2)
    - Clear listbox: `while (child := self._listbox.get_first_child()): self._listbox.remove(child)`
    - If not results: switch stack to "empty", return
    - For each station dict in results, create an `Adw.ActionRow`:
      - `row.set_title(s["name"])`
      - `row.set_subtitle(self._make_subtitle(s))`
      - Prefix: `Gtk.Image.new_from_icon_name("audio-x-generic-symbolic")` with `set_pixel_size(32)`
      - Suffix box (Gtk.Box horizontal, spacing=8):
        - Play button: `Gtk.Button.new_from_icon_name("media-playback-start-symbolic")`, tooltip "Preview"
          Connect clicked to `self._on_preview_clicked` passing the station dict
        - Save button: `Gtk.Button.new_from_icon_name("list-add-symbolic")`, tooltip "Save to library", add CSS class `suggested-action`
          Connect clicked to `self._on_save_clicked` passing the station dict and the save button ref
          If `s["url"] in self._saved_urls`: set insensitive, tooltip "Saved"
      - `self._listbox.append(row)`
    - Switch stack to "results"
    - Store `self._current_results = results` for reference

    **`_make_subtitle(self, s: dict) -> str`:**
    - Parts list. If `s.get("countrycode")`: append it. If `s.get("tags")`: split by comma, strip, take first 3, join with ", ", append. If `s.get("bitrate")`: append `f"{s['bitrate']}kbps"`.
    - Return `" \u00b7 ".join(parts)` (middle dot separator per UI-SPEC)

    **`_on_preview_clicked(self, btn, station_dict)`:**
    - Build a minimal `Station` for player: `Station(id=0, name=station_dict["name"], url=station_dict["url"], provider_id=None, provider_name=self._extract_provider(station_dict), tags=station_dict.get("tags", ""), station_art_path=None, album_fallback_path=None, icy_disabled=False)`
    - If `self._preview_station` and `self._preview_station.url == station_dict["url"]`:
      - Stop preview: `self.main_window.player.stop()`, set `self._preview_station = None`
      - Update button icon to `media-playback-start-symbolic`, tooltip "Preview"
      - Return
    - Stop any current preview: `self.main_window.player.stop()`
    - Reset all play buttons to start icon (iterate listbox children)
    - Set `self._preview_station = station`
    - Call `self.main_window.player.play(station, on_title=lambda t: None)` (ignore ICY titles during preview)
    - Update this button icon to `media-playback-stop-symbolic`, tooltip "Stop preview"

    **`_on_save_clicked(self, btn, station_dict)`:**
    - url = station_dict["url"]
    - If `self.repo.station_exists_by_url(url)`:
      - Show `Adw.MessageDialog(transient_for=self, heading="Station Already in Library", body=f'"{station_dict["name"]}" is already saved. No changes were made.')`
      - Add response "ok" with label "Got It", set as default and close response
      - `dlg.present()`, return (per D-09)
    - provider_name = `self._extract_provider(station_dict)`
    - tags = station_dict.get("tags", "").replace(",", ", ") — normalize spacing
    - `self.repo.insert_station(station_dict["name"], url, provider_name, tags)`
    - `self._saved_urls.add(url)`
    - btn.set_sensitive(False), btn.set_tooltip_text("Saved")
    - `self.main_window.reload_list()` (per D-10, DISC-04)

    **`_extract_provider(self, station_dict: dict) -> str`:** (per D-08, Research Pattern 4)
    - `network = (station_dict.get("network") or "").strip()`
    - If network: return network
    - `homepage = (station_dict.get("homepage") or "").strip()`
    - If homepage: `from urllib.parse import urlparse; domain = urlparse(homepage).netloc; return domain.replace("www.", "") or ""`
    - Return `""`

    **Close handling `_on_close_request(self, *_)`:**
    - `self._cancelled = True` (Pitfall 2)
    - Cancel debounce: `if self._debounce_id is not None: GLib.source_remove(self._debounce_id)` (Pitfall 1)
    - Resume prior station: if `self._preview_station is not None`: `self.main_window.player.stop()`. If `self._prior_station`: `self.main_window._play_station(self._prior_station)` (per D-06)
    - Return `False` (allow close)
    - Connect via `self.connect("close-request", self._on_close_request)`
