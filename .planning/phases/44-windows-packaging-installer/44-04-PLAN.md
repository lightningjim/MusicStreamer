<threat_model>
## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| installer download → user disk | User downloads MusicStreamer-2.0.0-win64-setup.exe from GitHub releases (HTTPS) |
| installer → install dir | Inno Setup writes files to %LOCALAPPDATA%\MusicStreamer (user-owned, no admin elevation) |
| installer → Start Menu | Shortcut registered under %APPDATA%\Microsoft\Windows\Start Menu\Programs\ (per-user) |
| build.ps1 → repo | Reads source files, version string, .spec; writes to dist/ (build-time only) |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-44-04-01 | T (Tampering) | DLL planting attack via writable install dir | mitigate | Install to %LOCALAPPDATA%\MusicStreamer (user-owned, no machine-wide write); no PATH manipulation; PrivilegesRequired=lowest. |
| T-44-04-02 | E (Elevation) | Installer requesting admin escalation users would routinely approve | mitigate | PrivilegesRequired=lowest in .iss prevents UAC prompt entirely (D-02). |
| T-44-04-03 | I (Information Disclosure) | Uninstaller deletes user data in %APPDATA%\musicstreamer | mitigate | NO [UninstallDelete] section — Inno Setup default uninstaller only removes what it installed (D-03). UAT D-21 item 2 verifies. |
| T-44-04-04 | T (Tampering) | Installer MITM during HTTPS download | accept | User downloads from GitHub releases over HTTPS; code signing deferred to v2.1+ (RESEARCH §Deferred). SmartScreen flags unsigned EXE — accepted friction for personal-scale. |
| T-44-04-05 | S (Spoofing) | Wrong AUMID in shortcut → SMTC shows "Unknown app" | mitigate | Pitfall 1 — both literals are exactly `org.lightningjim.MusicStreamer`; grep verification in acceptance criteria; case-sensitive match. |
| T-44-04-06 | T (Tampering) | AppId GUID change between versions breaks upgrade detection | mitigate | Pitfall 4 — `AppId={{914e9cb6-f320-478a-a2c4-e104cd450c88}` literal pinned, never changes. Comment in .iss documents the rule. |
| T-44-04-07 | I (Info Disclosure) | UPX-compressed GStreamer DLLs corrupt and expose memory | mitigate | `upx=False` in .spec preserved (Phase 43 anti-pattern). |
| T-44-04-08 | T (Tampering) | PKG-03 regression reintroduces console-flashing subprocess | mitigate | build.ps1 PKG-03 guard fails build on `subprocess.{Popen,run,call}` outside subprocess_utils.py (D-22). Belt-and-braces with the Python test from Plan 01. |
</threat_model>
