# Phase 88.3 Plan 02: Revert Conda WebEngine + B1 Drift-Guards Summary

**One-liner:** Removed all conda-side QtWebEngine assumptions (spec hiddenimports, `_run_check_webengine`, build.ps1 step-0 preflight + step-4e WEBENGINE GUARD) and re-pointed 5 G6 drift-guards at B1 invariants (WebEngine absent from conda artifact, separate `oauth_helper.exe` carries it).

## What Was Built

Under the spike-001-validated B1 architecture, WebEngine must NOT be in the conda main bundle (conda-forge ships zero PySide6 WebEngine bindings; layering pip PySide6-Addons causes the G6 DLL-load failure). This plan revertes every same-bundle assumption added by 88.3-01:

**Task 1 — Revert WebEngine from conda spec + __main__.py + build.ps1:**
- `packaging/windows/MusicStreamer.spec`: removed `"PySide6.QtWebEngineWidgets"` and `"PySide6.QtWebEngineCore"` hiddenimports plus the 12-line Phase 88.3 G6 comment block; added a one-line B1 comment pointing to `oauth_helper_standalone.spec`. `PySide6.QtNetwork` and `PySide6.QtSvg` preserved.
- `musicstreamer/__main__.py`: removed `_run_check_webengine()` function (29 lines), `--check-webengine` argparse argument, and `if args.check_webengine:` dispatch. `_run_check_mediakeys`, `_run_oauth_helper`, `--check-mediakeys`, `--oauth-helper` untouched.
- `packaging/windows/build.ps1`: removed step-0 PySide6-Addons preflight block (13 lines), `-SkipWebEngineGuard` param + comment (4 lines), step-4e WEBENGINE GUARD block (22 lines), and exit-codes header line `#  13=webengine not bundled...`. SMTC SMOKE GUARD (exit 11) and OAUTH HELPER GUARD (exit 12) untouched.

**Task 2 — Re-point 5 drift-guards at B1 invariants (TDD):**
- Rewrote `tests/test_packaging_webengine_spec.py` keeping module-scoped fixtures and `_strip_comments` helper but replacing all 5 assertion functions with B1-invariant negative gates:
  - G6-T1: spec has NO QtWebEngineWidgets/QtWebEngineCore on non-comment lines
  - G6-T2: PySide6.QtNetwork + PySide6.QtSvg still present (surgical revert check)
  - G6-T3: `__main__.py` has no `_run_check_webengine`/`--check-webengine`, still has `--oauth-helper`
  - G6-T4: `build.ps1` has no WEBENGINE GUARD/`--check-webengine`/`pyside6_webengine_missing`/SkipWebEngineGuard
  - G6-T5: `build.ps1` still has SMTC SMOKE GUARD (exit 11) + OAUTH HELPER GUARD (exit 12)
