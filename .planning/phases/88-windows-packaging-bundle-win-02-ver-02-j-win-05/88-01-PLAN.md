<objective>
Close WIN-02-A: add an explicit Inno Setup `[InstallDelete]` section to
`packaging/windows/MusicStreamer.iss` that removes the previous version's
Start-Menu `.lnk` BEFORE the new shortcut is created, and document the
"unpin from taskbar before upgrading" guidance in a new `RELEASE-NOTES.md`.

Purpose: A taskbar-pinned shortcut caches the AUMID at pin time (Pitfall 6).
Relying on Inno's AppId auto-uninstall is not guaranteed to clear a stale
`.lnk` before the new `[Icons]` entry lands, which can leave a pinned shortcut
pointing at a stale AUMID and break SMTC overlay identity on upgrade. An
explicit `[InstallDelete]` removes the old shortcut deterministically; the
release-notes footnote covers the residual taskbar-pin case the installer
cannot reach.

Output: edited `.iss` with an `[InstallDelete]` section; new `RELEASE-NOTES.md`.
</objective>
