## Typography

GTK4 uses CSS-based styling through the Adwaita theme. Adw.HeaderBar, Adw.ActionRow, Adw.StatusPage, and Gtk.Label all inherit system typography.

| Role | Size | Weight | Line Height |
|------|------|--------|-------------|
| Body (Gtk.Label) | 16px (system default) | 400 | 1.5 |
| Label (Gtk.Label subheadings) | 14px (via Adw small-body style) | 400 | 1.4 |
| Heading (Adw.StatusPage title) | 20px (Adw default) | 600 | 1.2 |
| Dialog title (Adw.HeaderBar) | 16px (Adw default) | 600 | 1.2 |

Apply `add_css_class("dim-label")` to secondary labels (e.g. "last imported" date). No custom CSS required beyond established patterns.

## Layout Contract

### Header Bar Menu (main_window.py)

- Add `Gtk.MenuButton` to the header bar with `icon-name="open-menu-symbolic"` (standard GNOME hamburger)
- Menu contains one item: "YouTube Cookies…" (`Gio.SimpleAction` → opens CookiesDialog)
- Menu button placed at end of `Adw.HeaderBar` (right side)

### Cookies Dialog (CookiesDialog)

- Widget class: `Adw.Window` (matches ImportDialog pattern — `set_transient_for`, `set_modal(True)`)
- Default size: 480 × 400
- Root: `Adw.ToolbarView` with `Adw.HeaderBar` top bar, title "YouTube Cookies"
- Content margins: 12px all sides (matches ImportDialog)

**Content layout (top to bottom):**

1. **Status row** — `Gtk.Label` showing either "No cookies imported" (dim-label) or "Last imported: {date}" (dim-label). Aligned left (xalign=0).

2. **File picker row** — `Gtk.Box` horizontal, spacing=8:
   - `Gtk.Label(label="Cookie file")` left-aligned, hexpand=False
   - `Gtk.Entry` read-only, showing filename or placeholder "No file selected", hexpand=True
   - `Gtk.Button(label="Browse…")` — opens `Gtk.FileChooserNative` filtered to `*.txt` files

3. **Other methods expander** — `Adw.ExpanderRow` with title "Other methods":
   - **Paste section**: `Gtk.TextView` with placeholder "Paste cookies.txt contents here…", min-height 80px, monospace font, vexpand=False
   - **Google login section**: `Gtk.Button(label="Sign in with Google")` — opens controlled browser window; replaces with spinner + "Signing in…" label while in progress

4. **Footer row** — `Gtk.Box` horizontal, spacing=8, packed at bottom:
   - `Gtk.Button(label="Clear")` with `destructive-action` CSS class, packed at start (hexpand=False)
   - spacer (`Gtk.Box` hexpand=True)
   - `Gtk.Button(label="Import Cookies")` with `suggested-action` CSS class, packed at end

**States:**

| State | What changes |
|-------|-------------|
| No cookies stored | Status label: "No cookies imported" (dim-label); Clear button insensitive |
| File selected (not yet imported) | Entry shows filename; Import button sensitive |
| Paste filled (not yet imported) | Import button sensitive |
| Importing (file or paste) | Import button insensitive, label "Importing…"; spinner shown inline next to button |
| Google login in progress | Sign-in button replaced by `Gtk.Spinner` + label "Signing in…" |
| Import success | Status label updates to "Last imported: {date}"; Import button label resets |
| Import error | Inline `Gtk.Label` with `error` CSS class below offending input; Import button re-enabled |
| Cookies stored | Status shows date; Clear button sensitive |

| Element | Copy |
|---------|------|
