<objective>
Add a dialog-level `_is_dirty()` predicate to `EditStationDialog` so that Plan 51-04 can intercept sibling-link clicks with a Save / Discard / Cancel confirmation when there are unsaved edits (D-11, D-12).

**Purpose:** The existing `EditStationDialog` has no change detection — `_is_new` (line 200) is a *lifecycle* marker, not a change detector. This plan introduces the project's first dirty-state mechanism and constrains its scope to exactly the six fields the user can edit: name, URL, provider, tags, ICY, streams table. The "new station" flag is orthogonal and must not be touched.

**Approach (planner's pick per D-12):** Snapshot-and-compare. At the end of `_populate`, capture a frozen baseline of all six fields' current values into `self._dirty_baseline`. `_is_dirty()` rebuilds the same shape from the live widgets and compares against the baseline. Rationale: robust against future field additions (one addition site — `_capture_dirty_baseline`); cheaper than wiring six separate `textChanged` / `currentTextChanged` / `stateChanged` / `cellChanged` listeners; survives the streams-table case cleanly (`QTableWidget` has no built-in dirty signal, so per-widget listeners would still need a row-snapshot fallback).

**Output:** Two new private methods on `EditStationDialog` + a `self._dirty_baseline` instance attribute, plus 8 new pytest-qt tests.
</objective>

    2. **Add helper `_snapshot_form_state(self) -> dict`** as a private method on `EditStationDialog` (place after `_populate`, around line 415). Implementation:

       ```python
       def _snapshot_form_state(self) -> dict:
           """Phase 51 / D-12: build a frozen snapshot of all six dirty-tracked fields.

           Used by _capture_dirty_baseline (called at end of _populate) and
           _is_dirty (called when navigating away). The snapshot is a dict
           that supports == comparison; nested containers are immutable
           (frozenset, tuple) so equality is deterministic regardless of
           insertion order.

           Scope (D-12): name, URL, provider, tags, ICY, streams table.
           Excludes: logo path (orthogonal — has its own refresh path), _is_new lifecycle flag.
           """
           tag_state = frozenset(
               (tag, chip.property("chipState") or "unselected")
               for tag, chip in self._tag_chips.items()
           )
           streams_snapshot: list[tuple[str, str, str, str, str]] = []
           table = self.streams_table
           for row in range(table.rowCount()):
               cells = []
               for col in (_COL_URL, _COL_QUALITY, _COL_CODEC, _COL_BITRATE, _COL_POSITION):
                   item = table.item(row, col)
                   cells.append(item.text() if item else "")
               streams_snapshot.append(tuple(cells))
           return {
               "name": self.name_edit.text(),
               "url": self.url_edit.text(),
               "provider": self.provider_combo.currentText(),
               "icy": self.icy_checkbox.isChecked(),
               "tags": tag_state,
               "streams": tuple(streams_snapshot),
           }
       ```

    3. **Add `_capture_dirty_baseline`** (placed immediately after `_snapshot_form_state`):

       ```python
       def _capture_dirty_baseline(self) -> None:
           """Phase 51 / D-12: freeze the current form state as the clean baseline.

           Called once at the end of _populate. Subsequent _is_dirty() calls
           compare the current snapshot against this captured baseline.
           """
           self._dirty_baseline = self._snapshot_form_state()
       ```

    4. **Add `_is_dirty`** (placed immediately after `_capture_dirty_baseline`):

       ```python
       def _is_dirty(self) -> bool:
           """Phase 51 / D-11 / D-12: return True if any tracked field has changed.

           Compares the live form snapshot against self._dirty_baseline. Returns
           False defensively when the baseline has not been captured yet
           (e.g. a caller invokes this before _populate has run).
           """
           if self._dirty_baseline is None:
               return False
           return self._snapshot_form_state() != self._dirty_baseline
       ```
