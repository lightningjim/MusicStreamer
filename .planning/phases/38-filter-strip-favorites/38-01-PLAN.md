## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-38-01 | I (Information Disclosure) | repo.py station favorites | accept | is_favorite is a boolean on local SQLite; no network exposure; same trust level as existing station data |
| T-38-02 | T (Tampering) | filter_utils search text | accept | Search text is used for in-memory string matching only; never reaches SQL (parameterized queries handle DB access separately) |
| T-38-03 | S (Spoofing) | ICY title in favorites | mitigate | ICY title rendered with Qt.PlainText (already enforced in NowPlayingPanel); parameterized INSERT in repo.add_favorite() prevents SQL injection |
