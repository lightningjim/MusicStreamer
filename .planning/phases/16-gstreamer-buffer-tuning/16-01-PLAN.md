<objective>
Set playbin3 buffer-duration and buffer-size properties to eliminate audible drop-outs on ShoutCast/HTTP streams.

Purpose: Default GStreamer buffering (2s / 2MB) causes rebuffering on high-bitrate streams during brief network hiccups. Raising to 5s / 5MB provides adequate headroom.
Output: Two constants in constants.py, two set_property calls in player.py, unit tests confirming correct values.
</objective>
