- [ ] CONSTRAINT: **`org.flatpak.Builder` rofiles-fuse bug** — When the system `flatpak-builder` binary is absent, `build.sh:51-54` falls back to `flatpak run org.flatpak.Builder`, whose nested bubblewrap sandbox cannot pass the FUSE fd to the host `fusermount` (`file descriptor 4 is not a socket`).
