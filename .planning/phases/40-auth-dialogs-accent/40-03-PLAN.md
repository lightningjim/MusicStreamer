Cookie validation pattern (from RESEARCH.md):
```python
def _validate_youtube_cookies(text: str) -> bool:
    for line in text.splitlines():
        if line.startswith("#") or not line.strip():
            continue
        parts = line.split("\t")
        if len(parts) >= 6 and ".youtube.com" in parts[0]:
            return True
    return False
```
