Event filter outline (MainWindow.eventFilter override) — must:
- Handle QEvent.MouseMove on obj is self.centralWidget()
- Convert event.position() to QPoint (Qt 6: position() returns QPointF, call .toPoint())
- In-zone gate: pos.x() <= _PEEK_TRIGGER_ZONE_PX AND (self._peek_overlay is None OR not self._peek_overlay.isVisible())
