**Layout (per UI-SPEC):** Non-modal QDialog, minimum 720x520.

**Search bar row** (QHBoxLayout, margins 16px, spacing 8px):
- `QLineEdit` placeholder "Search stations...", stretch 1. Return key triggers search.
- `QComboBox` tag filter — starts empty with placeholder "Loading..." until fetch_tags completes on daemon thread. First item after load: "" (All Tags).
- `QComboBox` country filter — same pattern with fetch_countries.
- `QPushButton("Search Stations")` — triggers `_start_search`.

**Progress bar:** `QProgressBar` below search bar. Hidden by default (`setVisible(False)`). During search: `setRange(0, 0)` for indeterminate, `setVisible(True)`. After results: `setVisible(False)`.

**Results table:** `QTableView` with a `QStandardItemModel`. Columns: Name (stretch), Tags (160px), Country (80px), Bitrate (80px), Play (60px), Save (60px). Read-only selection, single-row, `SelectRows`. `setAlternatingRowColors(True)`.

Per-row Play/Save: Add two extra columns. Play column: `QPushButton` with play/stop icon toggle. Save column: `QPushButton("Save to Library")` that disables after use. Use `setIndexWidget` to embed QPushButtons in the table.

**Worker threads:**
- `_TagWorker(QThread)`: calls `radio_browser.fetch_tags()`, emits `finished(list)`.
- `_CountryWorker(QThread)`: calls `radio_browser.fetch_countries()`, emits `finished(list)`.
- `_SearchWorker(QThread)`: calls `radio_browser.search_stations(name, tag, country)`, emits `finished(list)` or `error(str)`.

All workers parented to `self` for lifetime safety. Store as `self._tag_worker`, etc.

**Search flow (D-13):**
1. Disable search button, show progress bar (indeterminate), clear table.
2. Start `_SearchWorker` with current field values.
3. On `finished(results)`: populate table, hide progress bar, enable search button. Store results list as `self._results`.
4. On `error(msg)`: hide progress bar, enable search button, call `toast_callback(f"Search error: {msg}")`.

**Save flow (D-12):**
```python
def _on_save_row(self, row_index: int) -> None:
    result = self._results[row_index]
    stream_url = result.get("url_resolved") or result.get("url", "")
    if not stream_url:
        return
    self._repo.insert_station(
        name=result.get("name", "Unknown"),
        url=stream_url,
        provider_name="Radio-Browser",
        tags=result.get("tags", ""),
    )
    self._toast_callback(f"Saved '{result['name']}' to library")
    # Disable save button for this row
```

```python
def _on_play_row(self, row_index: int) -> None:
    result = self._results[row_index]
    if self._previewing_row == row_index:
        self._player.stop()
        self._previewing_row = -1
        # Update icon to play
    else:
        stream_url = result.get("url_resolved") or result.get("url", "")
        temp_station = Station(id=-1, name=result["name"], provider_id=None,

        temp_station.streams = [StationStream(id=-1, station_id=-1, url=stream_url)]
        self._player.play(temp_station)
        self._previewing_row = row_index
        # Update icon to stop
```

|----------|-------------|
| Radio-Browser API -> QTableView | Untrusted station names/tags from external API displayed in table cells |
| Radio-Browser URL -> Player | Untrusted stream URL passed to GStreamer pipeline |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-39-04 | Spoofing | Station name/tags from Radio-Browser in table | mitigate | Table cells use QStandardItem (plain text by default); any QLabel instances use `Qt.PlainText` |
| T-39-05 | Information Disclosure | Radio-Browser API queries | accept | Search queries sent to public Radio-Browser.info API; no PII involved |
| T-39-06 | Denial of Service | Large result set from search | mitigate | `limit=50` parameter caps result count; progress bar provides feedback during long searches |
</threat_model>
