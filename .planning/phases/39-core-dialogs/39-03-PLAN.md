`_YtScanWorker(QThread)`:
```python
class _YtScanWorker(QThread):
    finished = Signal(list)
    error = Signal(str)
    def __init__(self, url, parent=None):
        super().__init__(parent)
        self._url = url
    def run(self):
        try:
            results = yt_import.scan_playlist(self._url)
            live = [e for e in results if e.get("is_live") is True]
            self.finished.emit(live)
        except Exception as exc:
            self.error.emit(str(exc))
```

`_YtImportWorker(QThread)`:
```python
class _YtImportWorker(QThread):
    finished = Signal(int)  # count imported
    error = Signal(str)
    def __init__(self, entries, parent=None):
        super().__init__(parent)
        self._entries = entries
    def run(self):
        try:
            count = yt_import.import_stations(self._entries)
            self.finished.emit(count)
        except Exception as exc:
            self.error.emit(str(exc))
```

`_AaFetchWorker(QThread)`:
```python
class _AaFetchWorker(QThread):
    finished = Signal(list)
    error = Signal(str)
    def __init__(self, api_key, qualities, parent=None):
        super().__init__(parent)
        self._api_key = api_key
        self._qualities = qualities
    def run(self):
        try:
            channels = aa_import.fetch_channels_multi(self._api_key, self._qualities)
            self.finished.emit(channels)
        except ValueError as ve:
            self.error.emit(str(ve))
        except Exception as exc:
            self.error.emit(str(exc))
```

`_AaImportWorker(QThread)`:
```python
class _AaImportWorker(QThread):
    progress = Signal(int, int)  # current, total
    finished = Signal(int)
    error = Signal(str)
    def __init__(self, channels, api_key, qualities, parent=None):
        super().__init__(parent)
        self._channels = channels
        self._api_key = api_key
        self._qualities = qualities
    def run(self):
        try:
            count = aa_import.import_stations_multi(
                self._channels, self._api_key, self._qualities,
                on_progress=lambda cur, tot: self.progress.emit(cur, tot)
            )
            self.finished.emit(count)
        except Exception as exc:
            self.error.emit(str(exc))
```

CRITICAL: Do NOT pass any `Repo` instance to import workers. `aa_import.import_stations_multi` creates its own thread-local `Repo(db_connect())` internally (Pitfall 3 from RESEARCH.md).

<threat_model>
## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| YouTube playlist URL -> yt_import | User-supplied URL passed to yt-dlp library |
| AudioAddict API key -> aa_import | User-supplied key passed to external HTTP request |
| Import results -> QListWidget | Untrusted titles from external sources displayed in list |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-39-07 | Tampering | YouTube URL input | mitigate | `is_yt_playlist_url()` validates URL format before passing to yt-dlp; yt-dlp library handles URL parsing safely |
| T-39-08 | Information Disclosure | AudioAddict API key display | accept | echoMode Normal per D-16 — API key is not a high-sensitivity credential; displayed only in the dialog field |
| T-39-09 | Denial of Service | Large playlist scan | mitigate | Scan runs on daemon thread; UI remains responsive; progress bar provides cancellation feedback |
| T-39-10 | Spoofing | Imported station titles in QListWidget | accept | QListWidget items are plain text by default; no rich-text injection vector |
</threat_model>
