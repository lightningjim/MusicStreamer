**1. Create `musicstreamer/ui/accent_dialog.py`:**

Subclass `Adw.Window` (per project convention — all existing dialogs use Adw.Window, not Adw.Dialog). Constructor signature:

```python
class AccentDialog(Adw.Window):
    def __init__(self, app, repo: Repo, accent_provider: Gtk.CssProvider, main_window):
```

**Constructor setup:**
- `super().__init__(application=app, title="Accent Color")`
- `self.set_transient_for(main_window)`
- `self.set_modal(True)`
- `self.set_default_size(320, -1)`
- Store `self.repo`, `self.accent_provider`, `self._selected_btn = None`, `self._swatch_map = {}`
- `self._current_hex = repo.get_setting("accent_color", ACCENT_COLOR_DEFAULT).lower()`
- Call `self._build_ui()`

**_build_ui layout (vertical Gtk.Box):**

Use `Adw.ToolbarView` with an `Adw.HeaderBar` as top bar (matching project dialog pattern). Content is a vertical `Gtk.Box` with margin 16px all sides.

a) **Swatch grid** — `Gtk.FlowBox` with `set_max_children_per_line(8)`, `set_min_children_per_line(4)`, `set_selection_mode(Gtk.SelectionMode.NONE)`, `set_row_spacing(8)`, `set_column_spacing(8)`. For each hex in `ACCENT_PRESETS`:
   - Create a `Gtk.Button()` with `set_size_request(40, 40)`
   - Apply inline CSS via a per-button CssProvider: `button { background: {hex}; border-radius: 50%; min-width: 40px; min-height: 40px; }` — register at PRIORITY_APPLICATION
   - Store in `self._swatch_map[hex] = btn`
   - `btn.connect("clicked", self._on_swatch_clicked, hex)`
   - If hex matches `self._current_hex`, add `suggested-action` CSS class and store as `self._selected_btn`

b) **Separator** — `Gtk.Separator(orientation=Gtk.Orientation.HORIZONTAL)` with margin top/bottom 12px

c) **Hex entry** — `Adw.EntryRow(title="Hex color")`. Set text to `self._current_hex`. Connect `entry_row.connect("notify::text", self._on_text_changed)` to clear error. Connect `entry_row.connect("entry-activated", self._on_hex_submitted)` for Enter key. Also connect a focus controller: `focus_controller = Gtk.EventControllerFocus()`, `focus_controller.connect("leave", self._on_hex_focus_out)`, `entry_row.add_controller(focus_controller)`.

Set `self._hex_entry = entry_row` and add content box as child of toolbar_view. Set toolbar_view as the window content.

**_on_swatch_clicked(self, btn, hex_value):**
- Call `self._apply_color(hex_value)`
- Update hex entry text to `hex_value`

**_on_hex_submitted(self, entry_row):**
- `text = entry_row.get_text().strip().lower()`
- If not starts with "#", prepend "#"
- If `_is_valid_hex(text)`: call `self._apply_color(text)`
- Else: `entry_row.add_css_class("error")`

**_on_hex_focus_out(self, controller):**
- Same validation logic as `_on_hex_submitted`

**_on_text_changed(self, entry_row, *_):**
- `entry_row.remove_css_class("error")`

**_apply_color(self, hex_value):**
- `hex_value = hex_value.lower()`
- `self.accent_provider.load_from_string(build_accent_css(hex_value))`
- `self.repo.set_setting("accent_color", hex_value)`
- `self._current_hex = hex_value`
- Update swatch selection: remove `suggested-action` from `self._selected_btn` if set; if `hex_value` in `self._swatch_map`, add `suggested-action` to that button and store as `self._selected_btn`; else set `self._selected_btn = None`
