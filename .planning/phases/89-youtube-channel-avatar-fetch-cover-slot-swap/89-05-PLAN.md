## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| worker thread → Qt main thread | fetch result crosses the thread boundary; only via queued Signal |
| network → disk → DB | fetched avatar bytes are written to disk then a relative path to the DB column |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-89-13 | Denial of Service | worker raising / touching widgets off the main thread | mitigate | _AvatarFetchWorker.run never raises out (emits ("",token) on failure) and touches no widget; result marshalled via queued finished Signal; no QTimer.singleShot from the worker (spike landmine) |
| T-89-14 | Tampering | stale avatar from a superseded fast-typing fetch | mitigate | monotonic _avatar_fetch_token; _on_avatar_fetched discards results whose token != current (stale guard) |
