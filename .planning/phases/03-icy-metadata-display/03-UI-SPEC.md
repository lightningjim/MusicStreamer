# Phase 3 — UI Design Contract

> Visual and interaction contract for Phase 3: ICY Metadata Display.
> Generated by gsd-ui-researcher. Verified by gsd-ui-checker.

## Design System

| Property | Value |
|----------|-------|
| Tool | none — GTK4/Libadwaita desktop app |
| Preset | not applicable |
| Component library | Libadwaita (Adw) + GTK4 |
| Icon library | GNOME symbolic icons (Gtk.Image.new_from_icon_name) |
| Font | System default (Cantarell on GNOME) — do not override |

**Note:** Color, typography, and spacing are system-governed by Libadwaita's theme. No hex values or CSS tokens are declared here. Design decisions are expressed as widget choices and property settings. The 8-point scale below maps to GTK margin/spacing properties, not CSS. Inherits all conventions from Phase 2 UI-SPEC.

## Typography

Typography is fully system-controlled by Libadwaita. Do not set custom font sizes. Use Adw/GTK style classes only.

| Role | GTK Style Class | Usage |
|------|----------------|-------|
| Track title | `.title-3` | Current track title in now-playing panel center column |
| Station name subtitle | `.dim-label` | Station name displayed below track title in center column |
| Idle label | `.dim-label` | "Nothing playing" text in idle state |

Line height and weight are inherited from Libadwaita — do not override.

## Widget Inventory

Specific widget choices within Claude's discretion (from CONTEXT.md):

| Widget | Choice | Rationale |
|--------|--------|-----------|
| Now-playing panel container | `Gtk.Box(orientation=HORIZONTAL, spacing=8)` added via `shell.add_top_bar(panel_box)` | Third top bar in `Adw.ToolbarView` — consistent with filter strip pattern from Phase 2 |
| Station logo slot | `Gtk.Picture` sized 96×96, `set_content_fit(Gtk.ContentFit.COVER)` | `Gtk.Picture` handles aspect ratio and HiDPI naturally; `COVER` fills the square without letterboxing |
| Logo fallback | `Gtk.Image.new_from_icon_name("audio-x-generic-symbolic")` at icon-size 96 | Standard GNOME placeholder; symbolic scales with theme |
| Cover art placeholder | `Gtk.Image.new_from_icon_name("audio-x-generic-symbolic")` at icon-size 96 | Structurally identical to logo slot so Phase 4 can swap in `Gtk.Picture` with minimal changes |
| Center column container | `Gtk.Box(orientation=VERTICAL, spacing=4)` with `set_hexpand(True)` | Stacks track title, station name, and Stop button vertically in available space |
| Track title label | `Gtk.Label` with `.title-3` style class, `set_ellipsize(Pango.EllipsizeMode.END)` | Prevents overflow on long ICY title strings |
| Station name label | `Gtk.Label` with `.dim-label` style class, `set_ellipsize(Pango.EllipsizeMode.END)` | Secondary identity below track title |
| Stop button | `Gtk.Button(label="Stop")` moved from HeaderBar into center column | Relocated per CONTEXT.md locked decision |
| Panel background | No explicit color — inherits `@headerbar_bg_color` via `add_top_bar()` placement | Consistent with HeaderBar and filter strip visual weight |

## Interaction Contract

### Idle state (nothing playing)
- Now-playing panel is always visible at full 120px height
- Left slot: `audio-x-generic-symbolic` fallback icon
- Center title label: "Nothing playing" (`.dim-label` style — muted, not prominent)
- Center station name label: hidden (`set_visible(False)`)
- Stop button: insensitive (`set_sensitive(False)`) — visible but grayed out
- Right slot: `audio-x-generic-symbolic` placeholder icon

### Playing state (ShoutCast/AAC stream with ICY metadata) — NOW-01, NOW-02
- Left slot: station logo from `station.station_art_path` loaded as `Gtk.Picture`; fallback to symbolic icon if path is None or file missing
- Center title label: current track title from `Gst.TAG_TITLE` (`.title-3`) — updated via `on_title` callback
- Center station name label: `station.name` (`.dim-label`) — set once when playback starts, visible
- Stop button: sensitive (`set_sensitive(True)`)
- Right slot: `audio-x-generic-symbolic` placeholder (Phase 4 will fill this)
- When a new TAG message arrives with `Gst.TAG_TITLE`: call `on_title(title)` — lambda updates title label in-place via `set_text()`; no animation
- When TAG message has no `Gst.TAG_TITLE`: ignore — do not clear current title

### Playing state (YouTube stream, no ICY) — NOW-03
- `_play_youtube` already calls `on_title(station.name)` — no code change needed
- Title label shows station name (satisfies NOW-03)
- All other panel state identical to ShoutCast playing state

### Station logo loading — NOW-04
- On `_play_station`: load `station.station_art_path` into `Gtk.Picture.set_filename()`
- If `station_art_path` is None or `os.path.exists()` returns False: swap to symbolic icon fallback
- On `_stop()`: reset left slot to symbolic icon fallback; clear title to "Nothing playing"; hide station name label; set Stop button insensitive

### GStreamer TAG bus wiring
- `Player.__init__`: `bus.connect("message::tag", self._on_gst_tag)` alongside existing `message::error`
- `_on_gst_tag(bus, msg)`: extract `taglist = msg.parse_tag()`, read `taglist.get_string(Gst.TAG_TITLE)` — if present and `on_title` is set, call `self._on_title(value)`; if absent, return without action
